## 摘要
主流视觉-语言-动作（VLA）模型在马尔可夫假设下主要从当前观测预测动作，因此难以处理长时域、时间依赖的任务。现有的记忆增强型VLA要么扩展观测窗口，要么从记忆库中检索历史作为辅助策略侧上下文。然而，它们将记忆置于VLA推理的原生潜在嵌入空间之外，阻止了历史经验与多模态推理和动作形成的流畅交织。为此，我们引入了LaMem-VLA，一种原生潜在记忆框架，它将历史经验重构为潜在记忆令牌，并直接与VLA推理交织。其核心是四个协调组件：（i）一个策展者，将历史经验组织成互补的短期和长期记忆库；（ii）一个搜索者，利用多模态认知查询两个记忆库以检索上下文相关证据；（iii）一个压缩者，将检索到的证据重建为紧凑的短期和长期潜在记忆令牌；（iv）一个编织者，将这些记忆令牌与当前观测和指令注入到一个连续的嵌入序列中。通过完全在同一连续潜在空间中对历史经验进行表示、检索和消耗，LaMem-VLA使记忆能够直接参与VLA推理，并在有限上下文下指导动作生成。在SimplerEnv和LIBERO上的大量实验证明了我们的LaMem-VLA的优越性。

## Abstract
Mainstream Vision-Language-Action (VLA) models predict actions primarily from the current observation under a Markovian assumption, thus struggling with long-horizon, temporally dependent tasks. Existing memory-augmented VLAs either expand the observation window or retrieve history from the memory bank as auxiliary policy-side context. However, they leave memory outside the native latent embedding space of VLA reasoning, preventing historical experience from being fluidly interleaved with multimodal reasoning and action formation. To this end, we introduce LaMem-VLA, a latent-memory-native framework that reconstructs historical experience into latent memory tokens and directly interweaves them with VLA reasoning. At its core, LaMem-VLA introduces four coordinated components: (i) a curator that organizes historical experience into two complementary short-term and long-term memory vaults; (ii) a seeker that queries both vaults using the multimodal cognition to retrieve context-relevant evidence; (iii) a condenser that reconstructs the retrieved evidence into compact short-term and long-term latent memory tokens; and (iv) a weaver that injects these memory tokens with the current observation and instruction into one continuous embedding sequence. By representing, retrieving, and consuming historical experience entirely in the same continuous latent space, LaMem-VLA enables memory to directly participate in VLA reasoning and guide action generation under a bounded context. Extensive experiments on SimplerEnv and LIBERO demonstrate the superiority of our LaMem-VLA.

- **核心问题**：现有主流视觉-语言-动作（Vision-Language-Action, VLA）模型在马尔可夫假设下主要从当前观测预测动作，缺乏对历史状态转换、已完成步骤和任务进度的感知，导致在长时域（long-horizon）和时序依赖的机器人操作任务中表现不佳。
- **背景**：虽然已有工作通过扩展观测窗口或外部记忆库检索来增强VLA的时序意识，但这些方法要么计算成本随上下文长度增长，要么将历史记忆存储在VLA原生潜在嵌入空间之外，作为策略侧辅助上下文而非参与多模态推理的核心环节。这限制了记忆对动作形成的直接塑造能力。
- **研究动机**：将历史经验表示为**上下文原生的潜在记忆**，使其与当前观测、语言指令在同一连续潜在空间中进行存储、检索与消耗，从而无缝融入VLA的推理过程。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：提出**LaMem-VLA**，一个原生潜在记忆框架，将历史经验重构为短期和长期潜在记忆令牌，并通过四个协调模块将其直接交织到VLA推理序列中，最终由扩散动作专家生成动作。
- **关键技术细节**：
  - **Latent Memory Curator（潜在记忆策展者）**：将历史经验分解为两个互补记忆库：
    - **短期记忆库**（`M_short`）：存储视觉显著证据，每个单元为键值对（键是压缩视觉表示的均值池化，值是压缩后的视觉令牌）。
    - **长期记忆库**（`M_long`）：存储动作隐藏状态，跟踪任务进度和动作连续性。
    - 更新策略：当库容量超过上限`L`时，合并最相似的相邻单元以减少冗余。
  - **Latent Memory Seeker（潜在记忆搜索者）**：基于当前多模态认知表示`Qcon_t`构建上下文感知查询`Qt`，用余弦相似度从两个记忆库中分别检索Top-K个相关单元。
  - **Latent Memory Condenser（潜在记忆压缩者）**：将检索到的冗余证据通过轻量Transformer记忆形成器（memory former）和可学习记忆槽位，压缩为固定长度的短期记忆令牌`M_short`（长度`Ls`）和长期记忆令牌`M_long`（长度`Ll`）。
  - **Latent Memory Weaver（潜在记忆编织者）**：将压缩后的记忆令牌与当前观测令牌、指令令牌、动作查询令牌拼接，形成记忆增强的VLA输入序列`St`，使动作查询在自注意力中同时关注记忆、观测和指令。
  - **Diffusion-based Action Expert**：对内存增强后的动作令牌进行条件扩散去噪，生成连续7自由度动作块（动作块大小16）。损失函数为预测噪声的MSE。
- **公式/算法流程（文字说明）**：
  - 每个时间步，视觉-语言编码器生成视觉令牌`Xt`和指令令牌`I`。
  - Curator更新两个记忆库。
  - Seeker构建查询`Qt`，从`M_short`和`M_long`分别检索Top-K个单元，得到`Z_short`和`Z_long`。
  - Condenser通过记忆形成器`Fv`和`Fc`，将`Z_short`和`Z_long`分别压缩为`M_short`和`M_long`。
  - Weaver构建输入序列`St = [M_short; M_long; Xt; I; Qaction]`，通过VLM得到记忆接地动作令牌`Zaction`。
  - 扩散动作专家以`Zaction`为条件，迭代去噪生成动作序列。

### 7. 优点

- **范式创新**：首次将历史经验作为上下文原生的潜在记忆令牌，与多模态推理在同一潜在空间交融，超越了外部辅助记忆的局限。
- **双记忆互补设计**：短期视觉记忆与长期语义/动作连续记忆分别处理不同时间尺度的信息，更贴合长时域操作需求。
- **固定长度令牌压缩**：通过condenser将可变长度检索证据压缩为固定长度令牌，避免上下文膨胀，同时保持信息完整性。
- **计算效率**：使用轻量Transformer作为查询构建器和记忆形成器，额外开销有限；动作生成采用扩散模型保证平滑性。
