## 摘要
视觉-语言-行动（VLA）模型旨在将多模态输入映射到机器人动作。然而，大多数现有方法由于均匀处理所有视觉标记并使用人类选择的因素进行推理，缺乏强调任务关键证据的机制且忽略了潜在因素，因此难以覆盖复杂的动态场景。为解决此问题，我们提出LEEVLA，一种关注潜在环境演化中关键信息的VLA架构，它明确引导模型关注信息丰富区域，同时保留潜在世界表征的结构化演化。为识别显著且与指令相关的区域，我们引入漂移引导的动态优先级（DGDP），它结合动态位置优先级（DPP）和语义漂移引导（SDG），在训练过程中引导VLA智能体关注何处。在此基础上，我们引入结构化特征流生成（SFFG），通过原型到外围（P2P）预测对这些优先级特征如何在潜在空间中演化进行建模，以及一个互邻域对比（MC）损失来维持邻域间的拓扑一致性。DGDP和SFFG共同构成任务感知的“何处-如何”训练框架。在VLA基准上的大量实验表明，LEEVLA始终优于先前方法，证实了明确的任务证据引导和结构化潜在推理对于可扩展VLA都至关重要。我们的代码可在https://github.com/LyuQi127/LEEVLA获取。

## Abstract
Vision-language-action (VLA) models aim to map multimodal inputs to robot actions. However, most existing approaches struggle to cover complex dynamic scenarios due to treating all visual tokens uniformly and reasoning with human-selected factors, which lack mechanisms to emphasize task-critical evidence and ignore underlying factors. To address this issue, we propose LEEVLA, a VLA architecture for seeing what matters in Latent Environment Evolution that explicitly guides the model toward informative regions while preserving the structured evolution of latent world representations. To identify salient and instruction-relevant regions, we introduce drift-guided dynamic prioritization (DGDP), which combines dynamic position prioritization (DPP) with semantic drift guidance (SDG) to guide the VLA agent where to attend during training. On top of this, we introduce structured feature flow generation (SFFG), which models how these prioritized features should evolve in latent space via prototype-to-periphery (P2P) prediction, and a mutual-neighborhood contrastive (MC) loss to maintain topological consistency among neighborhoods. Together, DGDP and SFFG form a task-aware "where-how" training framework. Extensive experiments on VLA benchmarks show that LEEVLA consistently outperforms prior methods, confirming that explicit task-evidence guidance and structured latent reasoning are both crucial for scalable VLA. Our code is available at https://github.com/LyuQi127/LEEVLA.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：现有视觉-语言-动作（VLA）模型在处理复杂动态场景时存在两大缺陷：
  1. 对所有视觉标记（token）一视同仁，缺乏对任务关键区域（如机器人接触点、目标对象）的突出强调，导致监督信号被静态背景稀释。
  2. 通常依赖人类选择的特定上下文线索（如子目标图像、分割图、深度图）进行推理，限制了探索空间，忽略了许多未知但可能重要的潜在因素，且需要额外辅助模型增加计算开销。
- **动机**：作者主张直接在潜在特征空间中进行推理，将动作视为环境状态在高维空间中演化的驱动力，并引入显式的“何处关注”和“如何演化”机制，使模型能够自动发现任务关键的时空特征。
- **整体含义**：提出LEEVLA，通过**漂移引导动态优先级**（DGDP）指导模型聚焦何处，通过**结构化特征流生成**（SFFG）建模特征如何演化，从而在不增加推理开销的情况下显著提升VLA在多种基准上的性能。

## 2. 方法论：核心思想、关键技术细节

### 2.1 整体架构
- LEEVLA基于OpenVLA框架构建，使用预训练视觉编码器（DINOv2、SigLIP）提取视觉特征，经过投影器后与语言指令编码、本体感知嵌入一起送入大语言模型（LLM），最后由并行连续动作预测器输出动作块（action chunk）。
- 训练阶段引入两个额外模块：DGDP（何处关注）和SFFG（如何演化），推理阶段无额外开销。

### 2.2 漂移引导动态优先级（DGDP）
由两个子组件构成：

- **动态位置优先级（DPP）**：计算每个视觉patch在时间步t和t+T之间的余弦相似度变化，得到动态性得分θ_i = 1 - cos(v_{t,i}, v_{t+T,i})，得分高的区域（如机器人手、目标物体）获得更高权重。
- **语义漂移引导（SDG）**：计算每个patch与语言指令的相似度（指令相关得分r），并计算随时间变化的漂移Δ_i = clip((r_{t+T,i} - r_{t,i})/τ, -δ, δ)，再经归一化后通过指数函数得到语义漂移引导因子ω_i。最终优先级权重β_i = σ(ω_i·θ_i)（σ为sigmoid函数），使得模型关注那些动态性强且语义向指令方向演化的区域。

### 2.3 结构化特征流生成（SFFG）
- **原型到外围（P2P）预测**：对多视角观测的未来特征（t+T时刻）进行聚类，得到若干簇。在每个簇内按照特征到簇中心的欧氏距离排序（从近到远，即原型到外围）。P2P损失函数为：L_P2P = 1/|C| Σℓ 1/|→c_ℓ| Σ_j (α+β_j)·φ(ˆv_{t+T,j}, v_{t+T,j})，其中α为全局调制因子（保持对全局的关注），β_j为DGDP权重，φ为余弦嵌入损失。该损失鼓励模型按语义结构顺序预测未来特征。
- **互邻域对比损失（MC）**：为了在噪声聚类中稳定对比学习，构建一阶和二阶邻域集，仅保留互为邻居（mutual neighbors）的样本作为正样本，采用InfoNCE损失，剔除不对称或虚假连接。公式：L_MC = -1/|I| Σ_i (1/|G_i^+|) Σ_j log(exp(S_ij/τ_c)/Σ_{ℓ≠i} exp(S_iℓ/τ_c))，其中G_i^+为互邻域正样本集。

### 2.4 训练目标
总损失：L_total = λ1·L_action + λ2·L_P2P + λ3·L_MC，其中L_action为动作块L1回归损失，平衡三项的贡献。

## 3. 实验设计

### 3.1 数据集与基准
- **仿真环境**：LIBERO套件（包括Spatial、Object、Goal、Long四个子任务）和CALVIN benchmark（ABC-D长程任务）。LIBERO包含10个指令、50个episode，随机种子3个。CALVIN报告5个任务的成功率和平均完成长度。
- **真实环境**：通用机器人UR5机械臂（6自由度），执行放置物体、按压按钮、关闭抽屉三个任务，每个任务20次连续试验。

### 3.2 对比方法
- **小规模模型（<1B参数）**：Octo（0.1B）、UniACT（0.5B）、Seer（0.3B）、DreamVLA（0.3B）、FLOWER（1B）、LEEVLA-mini（0.5B）。
- **大规模模型（~7B参数）**：OpenVLA（7B）、π0（3B）、π0.5（3B）、OpenVLA-OFT（7B）、UniVLA（7B）、CoT-VLA（7B）、MemoryVLA、LEEVLA-large（7B）。

### 3.3 实验数量与充分性
- **LIBERO**：LEEVLA-large在Spatial (98.8%)、Object (99.0%)、Goal (98.6%)、Long (96.4%) 四个子任务均取得最优或接近最优平均98.2%成功率，显著超越之前的方法（如π0.5为97.5%，OpenVLA-OFT为97.1%）。
- **CALVIN**：LEEVLA-large在5个任务上平均完成长度4.34，优于π0.5（4.02）和OpenVLA-OFT（4.10）。
- **真实世界**：LEEVLA-large在Place、Press、Drawer三个任务上平均78.5%成功率，远高于OpenVLA的40%。
- **消融实验**：在LIBERO-Goal上逐步消融四个组件（P2P、MC、DPP、SDG），完整模型97.0%，单一组件缺失均导致下降，证明各组件互补。
- **额外分析**：验证了特征重排（P2P排序）的有效性（从94.7%提升至95.2%）；全局因子α的必要性（使用α时97.0%，不使用α时95.8%）；以及各组件计算开销（训练阶段SDG和MC耗时较高，但推理阶段与OpenVLA-OFT几乎相同延迟）。

## 6. 主要结论与发现

- **主要结论**：显式任务证据引导（DGDP）和结构化潜在推理（SFFG）均对可扩展VLA至关重要，两者共同作用可获得一致且显著的性能提升。
- **关键发现**：
  1. 在潜在特征空间中进行结构化预测（P2P排序）优于扁平化顺序预测，能保留语义连续性。
  2. 互邻域对比损失比普通对比损失更能抵抗聚类噪声。
  3. 利用动态性和语义漂移联合计算优先级比仅用动态性更好。
  4. 全局因子α防止模型过度关注动态区域而丢失背景上下文。
  5. 训练阶段的显式推理模块不增加推理时计算量和延迟，具有良好的实用性。

## 8. 不足与局限

- **实验覆盖**：真实环境仅测试了三个简单任务，缺乏复杂长程任务和高动态场景（如多物体交互）的评估。
- **泛化性**：虽在LIBERO和CALVIN上表现优异，但未在更多多样化数据集（如VIMA、MetaWorld）或跨具身形态（不同机器人）上验证。
- **消融设计**：消融实验仅在LIBERO-Goal单一子任务上进行，缺少在更多任务上的验证，可能掩盖组件在不同任务上的敏感性差异。
- **偏差风险**：模型可能仍偏爱训练数据中常见的视觉模式，对分布外场景（如未见过的物体颜色、材质）的鲁棒性未明确测试。
- **计算开销**：训练阶段的MC损失和SDG组件耗时较高（特别是MC需要构建邻域图，耗时1.2秒/批），可能限制大规模训练的速度。
- **方法论局限**：依赖预训练视觉特征的质量，若视觉编码器对目标任务不敏感，潜在空间的结构可能不可靠；P2P聚类数L为超参数，未讨论其敏感性。
