## 摘要
开放环境下的桌面机器人操作要求系统具备语义理解、精确几何姿态估计和高频动作生成能力。端到端的视觉-语言-动作（VLA）模型擅长语义泛化，但通常缺乏细粒度任务所需的显式几何约束，且训练成本高昂。为弥合高层语义与低层物理执行之间的鸿沟，我们提出OpenSPM——一种开放环境的空间持久记忆框架，由空间姿态记忆和流匹配动作生成模型组成。OpenSPM首先利用语义条件化3D感知和卡尔曼滤波持续跟踪6D姿态，然后从人类示范中提取关键空间姿态，将其保存为可迁移的、以物体为中心的空间持久记忆条目。推理时，OpenSPM根据自然语言指令检索相关记忆条目，通过SE(3)变换将空间姿态迁移至新场景，并借助轻量级条件流匹配模型生成高频动作块。结合实时本体感受状态反馈与终端残差校正，该系统有效抑制轨迹误差累积。在十项LIBERO-GOAL任务上的评估显示，OpenSPM达到85.6%的成功率，等效控制频率为1033.3 Hz，且推理所需AI算力极低。大量消融实验表明，结构化空间持久记忆与闭环残差校正在可靠的高频机器人操作中起关键作用。

## Abstract
Open-environment tabletop robotic manipulation requires systems to possess semantic understanding, precise geometric pose estimation, and high-frequency action generation. While end-to-end vision-language-action (VLA) models excel at semantic generalization, they often lack explicit geometric constraints for fine-grained tasks and require costly training. To bridge the gap between high-level semantics and low-level physical execution, we propose OpenSPM, an open environment spatial persistent memory framework consisting of spatial pose memory and flow-matching action generation model. OpenSPM first leverages semantically conditioned 3D perception and Kalman filtering to track continuous 6D poses. It then extracts key spatial poses from human demonstrations, keeping them as transferable, object-centric spatial persistent memory entries. During inference, OpenSPM retrieves relevant memory entries in terms of natural language instructions, transfers the spatial poses to new scenes using SE(3) transformations, and generates high-frequency action chunks via a lightweight conditional flow-matching model. Combined with real-time proprioceptive state feedback and terminal residual correction, the system effectively suppresses trajectory error accumulation. Evaluated on ten LIBERO-GOAL tasks, OpenSPM achieves an 85.6% success rate and an equivalent control frequency of 1033.3 Hz, while requiring minimal inference AI computing power. Extensive ablations illustrate that structured spatial persistent memory and closed-loop residual correction play a crucial role in reliable, high-frequency robotic manipulation.

# 2. 论文提出的方法论

## 2.1 核心思想
- 构建一个**开放环境空间持久记忆框架（OpenSPM）**，由两部分组成：
  1. **关键空间姿态记忆库**：从人类示范中提取阶段级（如接近、抓取、放置）末端执行器相对于物体的6D姿态，作为可迁移的几何先验。
  2. **闭环高频流匹配动作生成模型**：在相邻关键姿态间生成短时动作块，并利用实时状态反馈和终端残差校正抑制误差累积。

## 2.2 关键技术细节
- **语义条件3D感知与卡尔曼状态估计**：
  - 多视图语义分割 → 3D重建 → 卡尔曼滤波持续跟踪6D姿态，提供尺度一致、平滑的几何表示。
  - 状态向量包括平移、线速度、局部方向误差、角速度；观测协方差由分割置信度、掩码面积等多因素联合确定。
- **关键空间姿态记忆库构建**：
  - 从示范轨迹中提取关键姿态，依据：夹爪状态变化、末端速度/方向突变、以及多模态模型（Qwen3-VL）的语义判断。
  - 每个记忆项存储语义键、语言描述、末端相对物体姿态（ΔT<sub>o→e</sub>）、阶段标签等。
  - 核心：使用**物体坐标系相对姿态**而非世界坐标系绝对姿态，实现跨场景迁移。
- **相对姿态迁移（SE(3)变换）**：
  - 测试场景下，通过估计的新物体姿态乘以记忆中的相对姿态，得到迁移后的末端姿态。
  - 定理保证：若目标物体姿态经历同一刚体变换，局部交互几何不变。
- **条件流匹配动作生成**：
  - 以当前实际末端姿态、目标关键姿态、相对误差和段内进度作为条件，生成长度为5的动作块。
  - 训练：构建线性插值路径，学习速度场v<sub>θ</sub>，损失为MSE。
  - 推理：从噪声开始ODE积分得到动作块。模型仅240K参数，适合高频调用。
- **闭环执行与终端残差校正**：
  - 每段动作块开始时使用实时夹爪状态重新构建条件（段间闭环）。
  - 在抓取/放置关键节点附近，如果当前位置与目标姿态残差超过阈值，则重新生成校正动作块，直至收敛。

## 2.3 公式/算法流程（文字说明）
- 状态空间模型：状态预测 → 观测更新 → 轻量平滑。
- 记忆检索：融合语义匹配、几何一致性、历史成功率的多目标优化。
- 流匹配：构建噪声到数据的线性路径，训练速度场；推理时沿速度场积分。

# 7. 优点

- **高效的分解思路**：将高层语义（语言理解、经验检索）与底层几何执行（姿态估计、局部控制）解耦，大幅降低模型规模和推理成本。
- **极小的动作模型**：240K参数的流匹配模型即可实现高频（>1kHz）动作生成，显著优于大参数VLA模型。
- **强可迁移性**：通过物体坐标系的相对姿态存储和SE(3)迁移，支持跨场景零样本转移。
- **闭环鲁棒性**：段间状态反馈+终端残差校正有效抑制多段误差积累，使轻量模型也能可靠完成接触敏感操作。
- **消融实验严谨**：针对记忆、闭环、残差校正分别进行消融，清晰揭示各模块贡献。

# 8. 不足与局限

- **依赖初始感知质量**：上游分割、3D重建误差直接影响姿态估计和迁移准确性，对视觉相似、严重遮挡、细长物体表现较差。
- **仅在单一仿真benchmark验证**：未在真实机器人上部署测试，环境随机性、物理接触力等真实挑战未被充分覆盖。
- **训练算力未披露**：无法评估训练成本与效率，可能影响实用性和可复现性。
- **任务范围有限**：仅覆盖LIBERO-GOAL的10个桌面操作任务，未涉及更复杂的移动操作、灵巧手等。
- **失败模式分析**：部分失败源于环境几何约束（如路径被阻挡），残差校正无法克服；这表明仅靠局部姿态调整不足以应对全局空间冲突。
