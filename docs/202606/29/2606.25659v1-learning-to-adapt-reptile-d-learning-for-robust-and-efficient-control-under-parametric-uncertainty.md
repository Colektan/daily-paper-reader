## 摘要
基于学习的李雅普诺夫控制（LLC）为非线性系统提供了形式化的稳定性保证，但其有效性严重依赖于精确的系统模型。参数变化和不确定性可能使稳定性约束失效，导致代价高昂的重新训练。尽管D-学习可以在不依赖显式动力学模型的情况下估计李雅普诺夫导数，但它仍然受限于单任务动力学，并且在大的参数偏移下性能下降。我们提出Reptile-D-学习，一种利用Reptile元学习算法捕获不同参数系统间共享动力学结构的框架，从而学习可泛化的李雅普诺夫网络初始化和高性能控制器。在多个非线性控制系统上的实验表明，Reptile-D-学习显著提高了对未见参数配置的泛化能力和快速适应能力。

## Abstract
Learning-based Lyapunov Control (LLC) provides formal stability guarantees for nonlinear systems, but its validity relies heavily on accurate system models. Parameter variations and uncertainties may invalidate stability constraints, leading to costly retraining. Although D-learning can estimate Lyapunov derivatives without relying on explicit dynamics models, it remains limited by single-task dynamics and degrades under large parameter shifts. We propose Reptile-D-learning, a framework that leverages the Reptile meta-learning algorithm to capture shared dynamical structures across systems with different parameters, thereby learning a generalizable Lyapunov network initialization and a high-performance controller. Experiments on multiple nonlinear control systems demonstrate that Reptile-D-learning significantly improves both generalization and rapid adaptation to unseen parameter configurations.

### 2. 论文提出的方法论

- **核心思想**：将控制问题建模为一个双层优化问题。外层是元学习（Reptile），优化跨任务期望的适应后损失；内层是任务特定的 D-learning 适应。通过 Reptile 的一阶近似更新参数差，避免了 MAML 所需的高昂二阶 Hessian 计算。
- **关键技术细节**：
  - 联合元参数 Θ = {θ_V, ψ_D, φ} 包含三个网络：李雅普诺夫网络 V_θ(x)，D函数网络 D_ψ(x,u)（估计 Lyapunov 导数），控制策略 π_φ(x)。
  - 任务损失 L_Tξ = L_V + L_D + L_π，分别对应李雅普诺夫正定负定约束、D函数拟合、控制器使 D<0。
  - 内循环：在每个任务 ξ 上执行 K 步 D-learning 更新（交替优化 V、D、π），得到 Θ^K_ξ。
  - 外循环 Reptile 更新：Θ ← Θ + ε·(1/N) Σ_i (Θ^K_ξ_i - Θ)。该更新隐式包含二阶信息，但只计算参数差，内存效率高。
  - 从动力学分解角度分析了跨任务梯度一致性：在假设下，Reptile 的更新主要受共享动力学部分主导，有利于捕获跨任务共性。
- **算法流程**（文字描述）：
  1. 初始化元参数 Θ。
  2. 对每个元迭代，采样 N 个任务。
  3. 对每个任务，复制元参数 Θ_j = Θ，然后执行 K 步内循环：
     - 采集轨迹，更新 V（通过 L_V 梯度）、更新 D（通过 L_D 梯度）、更新 π（通过 L_π 梯度）。
  4. 收集所有任务的适应后参数 Θ^K_j。
  5. 计算 Reptile 元更新：Θ ← Θ + ε·(平均参数差)。
  6. 重复直到收敛。
