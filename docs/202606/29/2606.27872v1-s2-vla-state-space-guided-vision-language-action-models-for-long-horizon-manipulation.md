## 摘要
视觉-语言-动作（VLA）模型在机器人操作中展现出强大能力，但在长时程任务中，由于累积误差传播，其性能显著下降。这一局限主要源于静态特征融合机制——依赖固定权重组合视觉、语言和动作表征，导致模型无法适应任务执行的不同阶段。为解决此问题，我们提出S$^2$-VLA框架，引入态空间引导的自适应注意力（SSGAA）机制。SSGAA维护一个跟踪任务进度的信念状态，并生成动态门控权重，以自适应融合来自三个互补来源的信息：用于空间感知的视觉特征、用于高层任务规划的任务意图，以及用于执行一致性的时序动作序列。这种自适应融合使模型能够在整个任务执行过程中调整关注重点，与不同任务阶段不断变化的需求保持一致。尽管参数量仅为20亿，S$^2$-VLA持续优于更大的70亿级模型，并在长时程操作基准（包括LIBERO和SimplerEnv）上取得最先进性能，凸显了自适应特征融合对于长时程机器人操作的重要性。

## Abstract
Vision-Language-Action (VLA) models have demonstrated strong capabilities in robotic manipulation, but their performance degrades significantly in long-horizon tasks due to cumulative error propagation. This limitation largely arises from static feature fusion mechanisms that rely on fixed weights to combine visual, language, and action representations, preventing the model from adapting to different phases of task execution. To address this limitation, we propose S$^2$-VLA, a framework that introduces a State-Space Guided Adaptive Attention (SSGAA) mechanism. SSGAA maintains a belief state that tracks task progression and generates dynamic gating weights to adaptively fuse information from three complementary sources visual features for spatial perception, task intents for high-level task planning, and temporal action sequences for execution consistency. This adaptive fusion allows the model to shift its focus throughout task execution, aligning with the evolving requirements of different task stages. Despite its compact 2B parameter size, S$^2$-VLA consistently outperforms larger 7B-scale models and achieves state-of-the-art performance on long-horizon manipulation benchmarks, including LIBERO and SimplerEnv. highlighting the importance of adaptive feature fusion for long-horizon robotic manipulation.

## 2. 论文提出的方法论

### 核心思想
- 在VLA框架中引入**状态空间引导的自适应注意力（SSGAA）** 机制。核心是维护一个紧凑的**信念状态** \(b_t\)，该状态通过门控循环单元（GRU）递归更新，编码了任务执行进度与历史动作-感知信息。信念状态驱动一个门控网络，动态生成三个互补注意力分支的融合权重，这三个分支分别是：
    - **低层视觉交叉注意力**：提供精细空间定位。
    - **高层语义交叉注意力**：提供任务意图与规划。
    - **动作序列自注意力**：保证执行连贯性。

### 关键技术细节
1. **信念状态更新**  
   使用GRU处理过去的动作序列 \(A_{t-K:t-1}\) 和本体感受 \(P_t\)，输出隐藏状态，再线性投影得到信念状态 \(b_t\)。公式：  
   \((o_t^{(l)}, h_t^{(l)}) = f_\phi(h_t^{(l-1)}, A_{t-K:t-1}, P_t)\)  
   \(b_t^{(l)} = W_b \cdot o_t^{(l)} + \beta_b\)

2. **输入构造**  
   将视觉特征 \(F_t\)、语言指令标记、可学习的意图标记拼接为统一序列 \(X\)，经过Transformer编码器产生多层表征。

3. **SSGAA注意力机制**  
   - 三个并行分支：
     - 动作序列自注意力：作用于动作嵌入 \(A^{(0)} \in \mathbb{R}^{T \times K \times d}\)。
     - 低层视觉交叉注意力：对视觉标记隐藏状态做交叉注意力，保留空间细节。
     - 高层语义交叉注意力：对意图标记隐藏状态做交叉注意力，捕获任务语义。
   - **信念引导门控**：在每个SSGAA层，由信念状态 \(b_t\) 通过MLP和Softmax生成三个分支的动态权重 \(g_{vis}, g_{ite}, g_{act}\)，然后加权融合得到该层输出 \(H^{(l)}\)。

4. **并行动作解码**  
   最后使用LayerNorm和线性投影直接输出未来 \(K\) 步动作序列 \(\hat{A}_{t:t+K-1}\)。

5. **训练与推理**  
   - 端到端训练，损失函数为均方误差（动作预测）。
   - 推理时循环更新信念状态，动态调整门控权重，产生动作。

## 6. 论文的主要结论与发现

- S<sup>2</sup>-VLA在LIBERO上平均成功率达**98.2%**，在SimplerEnv上平均成功率达**78.1%**，均优于所有对比方法，包括7B规模模型。
- 尤其**在长时程子集（LIBERO Long）上**，S<sup>2</sup>-VLA（96.4%）显著超越其他方法（第二名CronusVLA为94.0%，OpenVLA-OFT为94.5%）。
- **消融实验核心结论**：
  - 动态门控能带来性能提升，但必须在**中间层**（最佳为第12层）使用；浅层或深层门控无效甚至有害。
  - **信念状态引导**是关键：若将其替换为静态向量，性能下降至95.8%（低于完整版96.4%），证明动态门控依赖于状态。
  - 多层组合门控反而恶化性能（94.4%），说明单点精确门控更好。
- **门控权重解释性**：可视化显示，模型在需要精确定位时（如接近目标）加大视觉注意力权重；在需要任务切换时（如抓取、放置）加大语义注意力权重；在稳定运动时保持动作自注意力主导。这验证了其**阶段自适应特性**。
- **推理效率**：S<sup>2</sup>-VLA达到**80.8 Hz**吞吐量，远高于其他7B方法（如OpenVLA-OFT为71.4 Hz，TraceVLA仅4.3 Hz）。
