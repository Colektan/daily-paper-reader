## 摘要
世界动作模型通过联合预测未来视觉状态和动作展示了有前景的机器人操作性能。然而，现有方法主要依赖短期历史和短期未来预测，这对于长时域任务而言是不够的，因为其正确执行依赖于早期的观测和任务进度。这类时间依赖任务需要有效利用互补的时间信息，包括近期局部上下文、跨阶段历史事件、即时未来动态和全局任务进度。为解决长期遗忘和对全局任务状态感知不足的问题，我们引入了DiM-WAM，一种记忆增强的世界动作模型，它整合了多尺度历史上下文、局部未来动态和全局任务进度。该记忆从真实观测中提取紧凑的视觉事件信息，通过独立的基于相似度的合并更新多个记忆库，然后读取带有库标识和时间嵌入的长期上下文，以调节视频和动作去噪。一个进度监督目标进一步促使记忆令牌不仅编码已完成的历史事件，还编码当前任务阶段及其对剩余任务的影响。在RMBench上，DiM-WAM将平均成功率从LingBot-VA的28.4%提升至69.8%，超过了显式记忆的Mem-0基线（42.0%）。在四个真实世界的Franka任务上，它将平均阶段成功率从70.7%提升至91.5%，全任务成功率从52.5%提升至80.0%。项目页面：https://wangkai-casia.github.io/dim-wam/。

## Abstract
World-action models have shown promising robot-manipulation performance by jointly predicting future visual states and actions. However, existing methods mainly rely on short-term history and short-horizon future prediction, which is insufficient for long-horizon tasks whose correct execution depends on earlier observations and task progress. Such temporally dependent tasks require effective use of complementary temporal information, including recent local context, cross-stage historical events, immediate future dynamics, and global task progress. To address long-term forgetting and poor awareness of the global task state, we introduce DiM-WAM, a memory-augmented world-action model that integrates multi-scale historical context, local future dynamics, and global task progress. The memory extracts compact visual event information from real observations, updates multiple memory banks through independent similarity-based merging, and then reads the bank-identity- and time-embedded long-term context to condition video and action denoising. A progress-supervision objective further encourages memory tokens to encode not only completed historical events but also the current task stage and its implications for the remaining task. On RMBench, DiM-WAM raises average success from 28.4% with LingBot-VA to 69.8%, exceeding the explicit-memory Mem-0 baseline at 42.0%. On four real-world Franka tasks, it improves average stage success from 70.7% to 91.5% and full-task success from 52.5% to 80.0%. # 论文总结：DIM-WAM: World-Action Modeling with Diverse Historical Event Memory

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：现有的世界动作模型（World-Action Models, WAMs）主要依赖短期历史窗口和短期未来预测，这导致在处理长时域（long-horizon）操作任务时性能显著下降。此类任务（如多步整理、装配、工具使用）具有强非马尔可夫性，其正确执行依赖于早期的观测、已完成的子任务以及当前任务进度。短期上下文无法提供跨阶段的证据，容易导致重复动作、跳过阶段或错误切换。
- **整体含义**：作者认为，长时域决策不应仅依靠增加窗口长度，而需要有机地组织四种互补的时间信息：近期局部历史、跨阶段历史事件、即时未来动态、全局任务进度。此前工作（如Mem-0、MemoryVLA等）主要针对动作预测，而WAM的记忆还需直接参与未来视觉状态的自回归生成，且生成误差可能污染后续历史。因此，提出一种专门为WAM设计的记忆增强方法，以在固定容量下有效管理长期上下文，并利用真实观测进行闭环校正。

## 2. 方法论：核心思想、关键技术细节、算法流程（文字说明）

### 核心思想
提出 **DIM-WAM**，一个记忆增强的世界动作模型。核心在于将长期历史显式组织为 **多类型历史事件记忆（Multi-Type Historical Event Memory）**，通过独立相似度合并的多记忆库存储跨阶段事件，并引入任务进度监督，使记忆不仅记录过去事件，还编码当前阶段及剩余任务信息。

### 关键技术细节

1. **多记忆库结构**：设有K个并行的固定容量记忆库，每个库含N个事件槽。不同库通过独立的压缩历史、不同的更新规则以及读时的库标识嵌入，学习互补的历史视图。
2. **视觉事件提取**：每个决策段，模型从真实观测中提取紧凑视觉事件令牌，然后每个库独立决定是否保留该事件。
3. **独立有界压缩（Similarity-based Merging）**：当库满时，根据语义相似度和时间距离计算合并优先级，合并最冗余的相邻历史事件对。合并采用质量加权平均。新事件仅当比当前最冗余历史对更具信息性时才会写入。
4. **合作式读取**：读取时，每个有效记忆令牌附加库标识嵌入，按时间顺序展平，并使用RoPE时间位置编码。记忆序列同时作为视频去噪和动作去噪的条件。
5. **任务进度监督**：从聚合的记忆表示预测离散的任务进度分布，使用交叉熵损失。该辅助目标促使记忆编码全局任务状态，而不需要显式的长时未来预测。
6. **训练损失**：视频预测损失 + 动作生成损失 + 多样性损失（防止不同库收敛到相同表示） + 进度监督损失。

### 算法流程（文字说明）
- **读-预测-交互-更新循环**：每个决策段，先读取长时记忆，结合局部上下文，联合预测未来视觉潜变量和动作块；执行动作后，接收真实观测，提取视觉事件，更新每个记忆库（独立相似度合并）。
- **训练时**：使用截断反向传播（长度4）以避免长序列的图过大；以0.3概率丢弃历史KV缓存条目，鼓励模型利用记忆令牌。
- **推理时**：记忆令牌仅作为当前段的临时注意条件，不扩展KV缓存；预测后清除临时K/V，持久记忆仅在收到真实反馈后更新。

## 6. 主要结论与发现

- **模拟性能**：DIM-WAM在RMBench上平均成功率达到69.8%，远超基座LingBot-VA（28.4%）和显式记忆基线Mem-0（42.0%）。尤其在M(1)任务上从22.8%提升至80.6%，M(n)任务上从35.5%提升至56.3%。
- **真实性能**：平均阶段成功率从70.7%提升至91.5%，全任务成功率从52.5%提升至80.0%。Find Blue Block任务全任务成功率从10.0%跃升至70.0%，Line Swap从60.0%提升至90.0%。Press Twice任务由于窗口已覆盖大部分轨迹，性能均为100%，表明长期记忆不损害短时控制。
- **消融结论**：多银行结构优于单银行（8×12 vs 1×32：96.5% vs 75.5%）。进度头移除导致平均成功率从96.5%降至90.5%，表明全局进度感知的重要性。
- **跨银行行为**：不同银行倾向于保留不同阶段的事件，形成互补表示，而非坍塌到同一记忆。

## 7. 优点

- **方法设计巧妙**：显式分解长期历史为多类型记忆库，通过独立合并和合作读取，避免了单库容量竞争与表示混杂。
- **闭环校正**：利用真实观测更新记忆，防止生成误差污染长期状态，同时联合调节视频和动作生成。
- **进度监督**：轻量级的辅助目标，无需显式长时未来预测，就能提供全局时间约束，提升任务一致性。
- **实验充分**：包含模拟和真实场景，多个基线对比，协议审计确保评估公平，消融和可视化深入。
