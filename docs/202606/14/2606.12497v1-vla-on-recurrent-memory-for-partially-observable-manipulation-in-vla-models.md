## 摘要
视觉-语言-动作（VLA）模型根据当前观察预测未来的动作块，但这种假设在部分可观察性下会失效，因为决策依赖于不再可见的信息。现有记忆增强的VLA同时引入了循环、检索、压缩模块、辅助目标、层次化记忆或特定任务的架构变更，因此循环本身的作用仍然与周围机制纠缠在一起。我们在一个强大的预训练VLA骨干中进行了循环的受控隔离研究。我们的公式通过一小组可学习的记忆标记来增强变换器，这些标记跨时间步传递并通过自注意力更新，使用随时间截断的反向传播进行端到端训练，无需辅助损失和架构变更。我们将其实例化为$μ$VLA，这是一系列OpenVLA-OFT变体，其参数由记忆宽度m、TBPTT长度K和记忆更新规则（跨步梯度或分离的EMA）决定，因此循环是唯一变化的因素。在MIKASA-Robo上，$μ$VLA在最强设置下将五个训练任务的平均成功率从0.42提高到0.84，并在保留任务上达到0.23，而基于无记忆的基线为0.07。在需要不同记忆结构的任务上，性能接近基线。在LIBERO上，最强的循环变体达到了96.2%的平均成功率，表明在全可观察性下没有退化。我们将这些结果解释为对骨干内最小循环能力范围的校准，识别了其足够的区域以及需要额外记忆结构的区域。演示和视频可以在https://avanturist322.github.io/mu-vla/找到。

## Abstract
Vision-language-action (VLA) models predict chunks of future actions from the current observation, an assumption that fails under partial observability, where decisions depend on information no longer visible. Existing memory-augmented VLAs simultaneously introduce recurrence, retrieval, compression modules, auxiliary objectives, hierarchical memory, or task-specific architectural changes, so the contribution of recurrence itself remains entangled with surrounding machinery. We present a controlled isolation study of recurrence in a strong pretrained VLA backbone. Our formulation augments the transformer with a small set of learnable memory tokens carried across timesteps and updated through self-attention, trained end to end with truncated backpropagation through time, with no auxiliary losses and no architectural changes. We instantiate this as $μ$VLA, a family of OpenVLA-OFT variants parameterized by memory width m, TBPTT length K, and the memory update rule (cross-step gradients or a detached EMA), so that recurrence is the only varying factor. On MIKASA-Robo, $μ$VLA improves average success rate on five training tasks from 0.42 to 0.84 at the strongest setting and reaches 0.23 on held-out tasks with the same memory structure versus 0.07 for the memoryless baseline. On tasks requiring different memory structure, performance remains near baseline. On LIBERO, the strongest recurrent variant achieves 96.2% average success, indicating no regression under full observability. We interpret these results as a calibration of the capability envelope of minimal in-backbone recurrence, identifying the regime in which it is sufficient and the regime where additional memory structure is required. Demos and videos can be found in https://avanturist322.github.io/mu-vla/.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 2.1 核心思想
在预训练VLA骨干（OpenVLA-OFT）的Transformer中插入少量可学习记忆标记（memory tokens），这些标记跨环境时间步通过自注意力前向传播进行更新，使用截断反向传播（TBPTT）端到端训练，不添加辅助损失或架构变更，仅将循环状态作为唯一变化因素。

### 2.2 关键技术细节

- **记忆标记与循环**：在输入序列中插入 m 个可学习记忆标记（位于PROPRIO和TEXT之间），维度为 d。初始记忆 M_init 为共享可学习参数，后续每步记忆从前一步的记忆位置隐藏状态读取。
- **注意力掩码守卫**：为防止记忆标记通过复制动作块到记忆的退化解，将上下文到动作块的注意力块置零，使记忆标记不能关注动作区域，但动作区域可以读取完整上下文。
- **Round-Robin 情节式数据加载器**：替代标准OpenVLA-OFT中混洗单时间步数据对的方式，维护 B 个独立流，每个流逐步遍历一个完整情节，并按固定顺序输出，确保循环状态在连续批次间正确传递。
- **TBPTT与EMA写入规则**：
  - TBPTT：积累 K 个连续步骤的L1块损失，然后进行一次反向传播，记忆在截断边界处分离。
  - EMA（指数移动平均）：使用分离的低通滤波器更新记忆，梯度仅本地单步。
- **后退视界推理**：每环境步重新查询模型，仅执行预测动作块的第一动作，保持记忆更新节奏与环境步一致，避免训练-测试不匹配。

### 2.3 公式/算法流程（文字说明）

- **训练损失**：在情节式数据上，每步执行一次前向传播，得到记忆更新和动作预测，积累损失 → 每K步做一次反向传播 → 分离记忆图。
- **EMA训练**：每步局部损失 + 反向传播 + 分离的EMA记忆更新。
- **推理**：每步使用当前观察、语言指令和上一步记忆进行前向，输出动作块，仅执行第一个动作。

1. **RQ1**：循环可在微调阶段添加到预训练VLA中，训练任务平均成功率从0.42提升至0.84（K=2），主要增益集中在线索回忆任务。

3. **RQ3**：最小循环在匹配记忆语义的留出任务上表现良好（0.23 vs 0.07），但在新记忆语义任务上接近基线（0.16 vs 0.19参考）。

5. **诊断**：记忆状态呈现阶段依赖的动态（阶段转换时记忆变化峰值），注意力rollout显示记忆关注线索区域；噪声干预证实循环信道被主动读取；最小循环对长时域依赖仍然脆弱。

## 7. 优点

- **受控隔离设计**：首次对VLA骨干中的循环进行严格隔离研究，仅改变循环状态，排除其他记忆机制的混杂。
- **简洁有效**：仅依靠少量可学习记忆标记+注意力掩码+情节式数据加载+TBPTT，无需辅助损失或架构变更，即可显著提升部分可观测任务性能。
- **全面诊断**：通过记忆干预、注意rollout、相位/OOD泛化等7种推理时分析，深入解释循环行为。
- **明确能力边界**：不仅报告提升，还量化了最小循环的适用区域和失效场景，为后续研究提供校准。
- **开源与可复现**：提供演示、视频和代码（基于MIT协议）。

## 8. 不足与局限

- **实验覆盖**：仅基于OpenVLA-OFT一个骨干，LoRA秩固定为32，结果可能受限于此；未测试更大骨干或全参数微调。

- **任务限制**：MIKASA-Robo为模拟器环境；部分可观测依赖结构由基准设计，真实世界部分可观测可能更复杂。

- **外部对比有限**：仅引用了CronusVLA和MemoryVLA在LIBERO上的已发表数据，未在MIKASA-Robo上与这些方法直接对比。
