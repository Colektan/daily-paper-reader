## 摘要
学习兼容表示旨在学习特征表示，使得当模型更新时，这些表示可以互换使用。本文证明，由d-Simplex固定分类器学习的静态表示蕴含了其正式定义中的兼容性。这一结果为未来工作奠定了基础，并可直接应用于实际学习场景中。我们解决了在模型顺序微调时使用d-Simplex固定分类器学习兼容性的挑战。使用交叉熵损失的d-Simplex固定分类器学习在训练时会对齐一阶统计量的特征分布，因此可能无法充分捕捉模型更新之间表示的高阶依赖性。为解决此问题，我们证明，通过交叉熵损失和对比度损失凸组合的方式训练d-Simplex固定分类器模型，不仅能够捕捉高阶依赖性，而且等价于在兼容约束下使用交叉熵损失学习。我们通过大量实验验证了我们的发现，还考虑了一个新场景：预训练模型被顺序微调，并偶尔被改进模型替换。我们证明，静态表示能够实现不间断的检索服务（无需重新处理图库图像），同时在模型更新和替换过程中提升性能，达到最先进水平。代码见 https://github.com/miccunifi/iamcl2r。

## Abstract
Learning compatible representations aims to learn feature representations that can be used interchangeably over time whenever a model undergoes updates. In this paper, we demonstrate that stationary representations learned by d-Simplex fixed classifiers imply compatibility as in its formal definition. This result establishes a foundation for future works and can be directly exploited in practical learning scenarios. We address the challenge of learning compatibility using $d$-Simplex fixed classifiers when the model is sequentially fine-tuned. Learning according to a d-Simplex fixed classifier with the cross-entropy loss aligns feature distributions at the first-order statistics. Consequently, it may not fully capture higher-order dependencies in the representation between model updates. To address this issue, we demonstrate that training the model using a $d$-Simplex fixed classifier through a convex combination of the cross-entropy loss and a contrastive loss not only captures higher-order dependencies, but is also equivalent to learning with the cross-entropy under the compatibility constraints. We confirm our findings with extensive experiments also considering a new scenario where a pre-trained model is sequentially fine-tuned and occasionally replaced with an improved model. We show that stationary representations enable uninterrupted retrieval services (without reprocessing gallery images) while improving performance during model updates and replacements, achieving state-of-the-art. ## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：在深度学习模型中，当模型更新（如增加新类别、微调）时，原有的特征表示会发生变化。为了维持检索等服务，通常需要重新计算所有图库（gallery）的特征，这在数据量巨大或隐私受限时成本极高甚至不可行。因此，需要学习**兼容表示**（compatible representations），使新旧模型提取的特征可以直接比较，避免重索引。
- **背景**：现有兼容表示学习分为两类：从头再训练和顺序微调。近期工作表明，使用**d-Simplex固定分类器**（将类原型固定为规则单形体顶点）学习的静态表示在两种方式下均表现出色，但其理论基础尚不完善。本文旨在补足这一理论空白，并解决顺序微调中交叉熵损失无法捕捉高阶依赖的问题。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：证明 d-Simplex固定分类器学习的静态表示（stationary representations）蕴含形式化的兼容性定义（Def.1）。在顺序微调场景中，为了捕捉模型更新间的高阶依赖，提出**高阶兼容性损失（HOC）**——交叉熵损失与对比损失的凸组合。
- **关键技术细节**：
  - **d-Simplex固定分类器**：将类原型 \( \mathbf{w}_1,\dots,\mathbf{w}_K \) 固定在单位超球面上构成规则单形体（Simplex-ETF），特征向量被约束为与对应原型对齐，保证类特征中心在模型更新后保持静止（stationary）。
  - **定理1（静态蕴含兼容）**：在UFM（Unconstrained Feature Model）假设下，利用d-Simplex固定分类器学到的类特征形成超球冠，模型更新后，同类超球冠更小（角度缩小），不同类超球冠仍保持最大分离。通过蒙特卡洛积分和经验估计证明，在期望上满足兼容性不等式。
  - **高阶兼容性损失 (LHOC)**：
    \[
    L_{\text{HOC}}(\phi_t) = \lambda L_{\text{SCE}}(\phi_t) + (1-\lambda) L_{\text{INCE}}(\phi_t,\phi_{t-1})
    \]
    其中 \( L_{\text{SCE}} \) 是带未来类回退的 d-Simplex交叉熵损失，\( L_{\text{INCE}} \) 是基于余弦相似度的对比损失（InfoNCE近似互信息）。该损失捕捉新旧表示分布间的高阶依赖，且等价于在兼容约束下的交叉熵学习（命题1）。
- **算法流程**（文字说明）：
  1. 预分配 K 个类原型（d-Simplex），固定不更新。
  2. 对每个任务 t，用当前任务数据（可能含经验回放缓冲区）训练模型 \(\phi_t\)。
  3. 损失函数为 LHOC：前项保持分类能力与未来类区域预留，后项通过对比学习（正样本对为同一图像在新旧模型下的特征）最大化新旧表示的互信息。
  4. 若发生模型替换（如用更大预训练模型），利用已固定的 d-Simplex 原型作为公共接口，新旧表示可直接比较。

## 6. 主要结论与发现

1. **理论证明**：d-Simplex固定分类器学习的静态表示在期望意义上满足兼容性定义，填补了理论空白。
2. **高阶兼容性损失的有效性**：HOC损失能捕捉模型更新间的高阶依赖，相比纯交叉熵或MSE蒸馏，在长时间序列微调中保持高兼容性和准确率。
3. **模型替换场景下的优势**：仅d-Simplex类方法（d-Simplex-FD和d-Simplex-HOC）能在模型替换后无缝利用新模型能力而不丢失兼容性，其他方法性能急剧下降。d-Simplex-HOC因高阶对齐表现最佳。
4. **消融结论**：预分配类数对性能影响不大；λ=0.1、ρ=5~10为较好设定；学习率需折中；经验回放有益但在无回放时仍可工作；训练样本量足够时性能稳定。
