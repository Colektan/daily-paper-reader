## 摘要
标准集合表示学习方法通常在精心整理的数据上表现优异，但往往忽略了推理时元素损坏的挑战。这指的是部署模型遇到元素级别的退化（如异常值或缺失组件）时，可能会扭曲集合表示并降低性能。我们提出了SW-DRSO，一个专为集合定制的分布鲁棒优化框架。SW-DRSO并非仅最小化观测训练数据上的损失，而是在一组合理的推理时变体上优化最坏情况期望损失的可处理替代项。我们引入了一个重心对抗者，通过可微的训练时优化对单纯形权重进行近似，从而逼近对损坏集合的棘手搜索。跨四个任务的大量实验表明，SW-DRSO能够在保持高整体性能的同时，有效增强对损坏的鲁棒性。

## Abstract
Standard Set Representation Learning methods typically excel on curated data but often overlook the challenge of inference-time element corruption. This refers to scenarios where deployed models encounter element-level degradations, such as outliers or missing components, that may distort set representation and degrade performance. We propose SW-DRSO, a distributionally robust optimization framework tailored for sets. Rather than minimizing loss solely on observed training data, SW-DRSO optimizes a tractable surrogate of the worst-case expected loss over a family of plausible inference-time variations. We introduce a barycentric adversary that approximates the intractable search over corrupted sets by a differentiable training-time optimization over simplex weights. Extensive experiments across four tasks demonstrate that SW-DRSO effectively enhances robustness against corruption while maintaining high overall performance.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：标准集合表示学习（SRL）方法在整理好的数据上表现良好，但实际部署时，模型会遭遇**推理时元素损坏**（inference-time element corruption）——即输入集合中部分元素出现退化（如异常值、缺失），导致集合表示失真、下游任务性能严重下降。这种损坏通常是稀疏的、标签保持的，但即使少量异常元素也能通过注意力池化等机制扭曲全局表示。
- **研究动机**：现有SRL方法主要依赖经验风险最小化（ERM），对推理时的分布偏移缺乏鲁棒性。分布鲁棒优化（DRO）虽能对最坏情况分布优化，但直接应用于集合数据面临两大挑战：1）集合离散、组合性，难以定义合理的模糊区域；2）内层最大化需搜索指数级组合空间，计算不可行。
- **整体目标**：提出一个可扩展的DRO框架SW-DRSO，利用切片Wasserstein（SW）度量定义模糊区域，并通过重心对抗者将棘手搜索转化为可微优化，同时保持集合表示的置换不变性和基数适应性。

### 2. 论文提出的方法论

- **核心思想**：将集合建模为经验分布，通过SW度量衡量集合间距离；在SW球内定义可行腐蚀区域，然后优化最坏情况损失；为求解内层最大化，引入Wasserstein重心合成策略，将搜索空间从离散集合转换为低维单纯形权重。
- **关键技术细节**：
  1. **SW模糊区域定义**：集合S表示为经验测度μS，腐蚀区域Γ(S) = {S' : SW(μS, μS') ≤ ρ}，SW距离通过随机投影方向近似。
  2. **Wasserstein-aware编码器**：采用PSWE（Naderializadeh et al., 2021），通过学习参考集O，利用1D最优运输映射将集合元素映射到固定长度的编码v = fSW(S)，使得编码空间中的欧氏距离近似于SW距离。
  3. **Wasserstein重心对抗者**：对每个训练样本S，从当前批次中通过K近邻构建局部邻居池G(S) = {Sk}（基于编码距离）。合成嵌入¯v_S(Λ) = ∑ λ_k v_Sk，其中Λ属于概率单纯形ΔK。该合成嵌入对应S的Wasserstein重心，且满足SW约束。内层最大化转化为在ΔK上最大化损失ℓ(g(¯v_S(Λ)), y)。
  4. **优化流程**：采用投影梯度上升（T步）求解内层最大化，初始化Λ均匀分布，更新Λ后投影到ΔK。外层面定参数f,g通过标准梯度下降优化组合损失（ERM损失 + α × 鲁棒损失）。总算法如Algorithm 1所示。

- SW-DRSO在**所有任务和几乎所有评估子集**（干净/轻度/重度）上均取得最佳或接近最佳的性能，尤其在重度腐蚀场景下优势显著（例如Task I LiveJ上Recall@10相对PSWE提升4.42%）。

- 相比其他DRO变体（WDRO, KL-DRO, MMD-DRO），SW-DRSO取得最佳精度-效率折衷，归因于SW编码器与SW模糊区域的几何对齐。
