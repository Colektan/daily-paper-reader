## 摘要
使用强化学习（RL）训练的大型语言模型智能体通常学习脆弱的、特定于任务的捷径。我们假设，当智能体的成功轨迹在结构上可压缩，分解为一小组可重用的抽象模式时，它们能更好地泛化。为形式化这一观点，我们引入ReuseRL，它将智能体RL建立在最小描述长度（MDL）原则之上。ReuseRL从成功轨迹中提取共享的技能字典，并通过分段成本增强RL目标，显式惩罚编码不良的特异行为。我们证明了该压缩惩罚的PAC-Bayes泛化界。在ALFWorld、TextWorld-Cooking和Countdown-Stepwise上，ReuseRL在分布内和分布外任务上的成功率均优于vanilla GRPO和强大的回合长度基线。

## Abstract
Large language model agents trained with reinforcement learning (RL) often learn brittle, task-specific shortcuts. We hypothesize that agents generalize better when their successful trajectories are structurally compressible, decomposed into a small set of reusable abstract patterns. To formalize this, we introduce ReuseRL, which grounds agentic RL in the Minimum Description Length (MDL) principle. ReuseRL extracts a shared skill dictionary from successful trajectories and augments the RL objective with a segmentation cost, explicitly penalizing idiosyncratic behaviors that encode poorly. We prove a PAC-Bayes generalization bound for this compression penalty. Across ALFWorld, TextWorld-Cooking, and Countdown-Stepwise, ReuseRL improves in- and out-of-distribution success over vanilla GRPO and strong round-length baselines.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：大型语言模型（LLM）智能体在使用强化学习（RL）训练时，常学会脆弱的、任务特定的捷径（brittle shortcuts），导致泛化能力差。这种现象被称为“推理崩溃”（reasoning collapse），即过度优化放大了记忆化的行为模式，而牺牲了可迁移的推理能力。
- **核心假设**：智能体泛化更好的关键是其成功轨迹在结构上具有可压缩性——即能够被分解为一小组可复用的抽象模式（技能）。作者将这种属性称为“结构可压缩性”，并认为它与最小描述长度（MDL）原则紧密相关。
- **整体含义**：通过直接惩罚那些编码低效的、特异性的成功行为，并奖励那些在共享技能字典下可压缩的行为，可以提升智能体的泛化性能。这为缓解RL训练中的推理崩溃提供了新的理论视角和实用方法。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 核心思想
- 将智能体RL建立在**最小描述长度（MDL）**原则之上：最优行为是一组可复用技能的最短编码。
- 通过**EM式交替优化**：E-step从成功轨迹集中提取共享技能字典（最小化批级MDL损失），M-step在GRPO目标中增加基于该字典的分段成本惩罚，引导策略生成更可压缩的轨迹。

### 关键技术细节
- **技能投影**：定义原子技能字母表Σ（每个环境5~26个），通过基于动词的规则映射将原始动作轨迹投影为技能序列s∈Σ*。
- **技能字典**：包含共享的可复用于序列（短语），长度上限L=4。字典必须包含所有原子技能（确保任何轨迹均可编码）。
- **分段成本（SegCost）**：对于给定字典C，一个技能序列s的最小分段数seg(s,C)。SegCost(s|C)=seg(s,C)/T，值越小表示越可压缩。
- **批级MDL目标**：平衡字典存储成本与每个轨迹的编码成本。通过贪心BPE合并近似求解最优字典。
- **RL目标增强**：对每个成功轨迹的原始奖励减去λ · SegCost，失败轨迹不变。用GRPO更新策略。

### 算法流程（文字说明）
1. 策略πθ在批量任务上采样轨迹，获得成功轨迹集合B+。
2. 对每个成功轨迹，通过规则映射计算技能序列s。
3. （可选）将成功技能序列存入全局FIFO缓冲区G。
4. E-step：对B+∪G中的技能序列，通过贪心BPE合并（最小化批MDL）提取新字典C。
5. M-step：对每个成功轨迹计算分段成本，构建形状奖励；使用GRPO更新策略πθ。
6. 重复直至收敛。

## 6. 主要结论与发现

1. **结构压缩优于简单长度压缩**：ReuseRL（分段成本）在三个基准上均优于纯回合长度惩罚，尤其在中需要处理不可逆失败的环境（如TextWorld-Cooking）中，纯长度惩罚导致灾难性烧焦失败（74%失败率），而ReuseRL仅3.3%。
2. **泛化增益显著**：在ALFWorld OOD分片上，ReuseRL达到93.28% SC@7，比vanilla GRPO提升13.43个百分点，比纯长度惩罚提升1.49个百分点。
3. **压缩信号对抗推理崩溃**：ReuseRL有效减少无效动作、动作重复、冗余导航等现象，形成更紧凑、可复用的行为模式。
4. **全局缓冲区效果依赖任务**：在ALFWorld和Countdown-Stepwise上有益，但在TextWorld-Cooking上可能因过度代表简单模式而损害硬分集表现。
5. **理论界支持实践**：PAC-Bayes界表明经验MDL目标可以可靠衡量行为结构性，为惩罚提供理论保障。
