## 摘要
视觉-语言-动作（VLA）模型已成为具身智能的重要范式。然而，现有的VLA模型通常假设光照良好且稳定的室内环境，而现实世界中的具身操作可能涉及光照变化导致的退化RGB观测，这给鲁棒的机器人操作带来了关键挑战。为弥补这一不足，我们提出Event-VLA，一种事件增强的VLA框架，用于在不同光照条件下实现可泛化的操作。我们将退化可见性下的基于VLA的操作形式化为以RGB为中心策略的实用鲁棒性问题，并引入事件流作为一种光照鲁棒、运动敏感的互补观测，以提升跨可见性水平的鲁棒性。具体而言，与直接将事件特征合并到全局语义标记空间中的传统多模态融合不同，Event-VLA通过动作查询路由路径注入事件信息。它使用可学习的动作查询从VLA推理过程中提取与任务相关的语义，并通过门控交叉注意力选择性地聚合事件标记，以构建事件感知的动作表示。这种设计保留了预训练的RGB-语言语义先验，同时有效利用事件信息实现鲁棒的动作预测。仿真和真实世界部署实验表明，Event-VLA在正常光照下保持强大的操作性能，并在低光照退化及接近黑暗的真实场景中提高了成功率。

## Abstract
Vision-Language-Action (VLA) models have become an important paradigm of embodied AI. However, existing VLA models typically assume well-lit and stable indoor settings, while real-world embodied manipulation may involve degraded RGB observations caused by illumination shifts, posing critical challenges for robust robotic manipulation. To address this gap, we propose \textbf{Event-VLA}, an event-enhanced VLA framework for generalizable manipulation across varying illumination conditions. We formulate VLA-based manipulation under degraded visibility as a practical robustness problem for RGB-centric policies, and introduce event streams as an illumination-robust, motion-sensitive complementary observation to improve robustness across visibility levels. Specifically, unlike conventional multimodal fusion that directly merges event features into the global semantic token space, Event-VLA injects event information through an action-query routing pathway. It uses learnable action queries to extract task-relevant semantics from the VLA reasoning process, and selectively aggregates event tokens via gated cross-attention to construct event-aware action representations. This design preserves the pretrained RGB-language semantic priors while effectively leveraging event information for robust action prediction. Experiments in simulation and real-world deployment show that Event-VLA maintains strong manipulation performance under normal lighting and improves success rates under low-light degradation and near-dark real-world settings.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有视觉-语言-动作（VLA）模型大多假设理想、光照稳定的室内场景，但在真实世界中，光照变化（如低光照、近黑暗）会导致 RGB 观测退化（纹理丢失、信噪比下降、运动模糊），从而破坏语义基础，引发不稳定的动作预测。缺乏对退化可见性的鲁棒性。
- **研究意义**：将退化可见性下的 VLA 操作形式化为以 RGB 为中心策略的鲁棒性问题，并引入**事件相机**作为补充感知。事件相机具有高动态范围、低延迟、对运动敏感且光照鲁棒的特性，能够捕获 RGB 在低光照下丢失的运动和边缘线索。
- **整体含义**：事件增强可提升 VLA 在跨光照条件下的泛化能力，但关键问题是如何将异步事件流有效集成到预训练 VLA 中而不破坏其原始先验。Event-VLA 通过**动作条件查询路由（Action-Query Routing）** 实现事件的选择性注入，在不干扰 RGB-语言语义的前提下提升鲁棒性。

## 2. 论文提出的方法论

### 核心思想
- 语义上下文由预训练 VLA 建模，事件信息应以受控方式注入动作通路，而非融合到全局语义令牌空间。
- 使用可学习动作查询从 VLA 推理中提取任务相关语义，并通过门控交叉注意力（Gated Cross-Attention）选择性地聚合事件令牌，形成事件感知的动作表示。

### 关键技术细节
#### （1）**PREI（Physical Residual Event Integration）**
- 将异步事件流压缩为三通道物理残差图：
  - **瞬时通道（Instantaneous）**：近期局部变化，短衰减常数。
  - **显著通道（Salient）**：局部突出活动，经空间平滑归一化。
  - **持久通道（Persistent）**：窗口内累积事件计数。
- 归一化至 [0,1]，形成紧凑、与 VLA 兼容的表示。
- 事件编码器通过特征蒸馏（从冻结的 DINO 视觉教师蒸馏）提升与 VLA 隐藏空间的兼容性。

#### （2）**动作条件事件接口（Action-Conditioned Event Interface）**
- VLA 输入：RGB 视觉令牌 + 语言令牌 + 本体感觉令牌 + 动作占位符 + **可学习查询令牌**（公共查询、动作查询、事件查询）。
- 预训练 VLA 骨干（Prismatic-7B, 初始化自 OpenVLA）生成上下文隐藏状态。
- **事件令牌不送入 VLA 骨干**，而是通过后骨干门控交叉注意力与 VLA 隐藏令牌融合：
  - 交叉注意力生成事件感知特征，再经门控（sigmoid）与残差连接。
  - 查询引导路由（Query-guided Token Routing）将融合令牌分离为动作相关和事件相关表示。

#### （3）训练目标
- 主要目标：动作预测 L1 损失（动作块大小 K=8）。
- 辅助正则化：
  - 未来 PREI 重建损失（带内容掩码）。
  - 一阶导数一致性损失（鼓励边缘结构保持）。
- 总损失：L = L_action + λ_evt L_evt + λ_deriv L_deriv。辅助头仅在训练时使用。

## 6. 论文的主要结论与发现

- **主要结论**：Event-VLA 在正常光照下保持与 RGB 基线相当的性能（表1，平均 96.5% SR），并在光照退化下显著超越所有基线（LL-Severe 平均 95.6% SR vs 最佳 RGB 基线 69.6%）。动作条件事件接口优于直接统一编码或适配器融合，且添加延迟极小（约 2.16 ms）。
- **关键发现**：
  - **事件表示**：PREI 优于时间表面，重建误差更低且下游成功率更高。
  - **接口设计**：查询路由（Query Routing）获得最佳 SR 与低延迟；统一编码虽性能接近但延迟过高。
  - **查询消融**：公共查询 + 动作查询 + 事件查询组合最佳。
  - **正则化**：未来事件预测辅助损失（特别是导数一致性）带来稳定提升。
  - **可移植性**：训练后禁用事件通路（Ours w/o event）性能几乎不变，证明不破坏原有 VLA 先验。

## 7. 优点

1. **新颖的问题设定**：首次明确将光照退化下的 VLA 操控作为事件- VLA 接口问题，并系统化研究。
2. **设计优雅**：事件信息不进入预训练骨干，通过后骨干门控交叉注意力与查询路由注入，保留 RGB-语言先验，避免灾难性遗忘。
3. **高效的表示层（PREI）**：将异步事件压缩为物理残差图，兼顾瞬时、显著、持久线索，与 VLA 兼容且可蒸馏对齐。
4. **低延迟**：推理仅增加约 2.16 ms（RTX 4090），适合实时操控。
5. **充分的消融与分析**：覆盖表示、接口、查询、正则化四个维度，揭示了关键设计选择。
6. **强鲁棒性**：在重度退化（LL-Severe）下仍维持 95.6% 成功率，远超 RGB 基线（最高 69.6%），在真实近黑暗场景也提升明显。

## 8. 不足与局限

1. **模拟器依赖**：LIBERO-Cross 使用 RGB-to-Event 模拟器（基于 v2e 和 U-Net），可能不完全反映真实事件相机的噪声、触发动态、校准误差。尽管作者做了验证，但真实事件质量可能存在差距。
2. **真实评估规模有限**：仅 4 个任务，每条件 10 次试验；未涵盖更多物体、环境变化、长期任务。结论泛化性有待验证。
3. **仅针对光照退化**：未测试其他常见退化（如遮挡、运动模糊、传感器损坏），事件对非运动场景的辅助作用有限。
4. **硬件要求**：需要额外事件相机和同步，增加部署复杂度与成本；同步误差可能影响性能。
5. **事件表示损失**：PREI 为极性不可知设计，丢弃了亮度增减方向信息，可能在某些场景下丢失细节。
6. **训练成本高**：每子基准需 8× H100 约 140k 步，计算资源需求较大。
