## 摘要
我们研究动作条件世界建模作为可扩展的学习可迁移动力学先验的方法，用于机器人学习。通过预训练一个模型来预测动作如何驱动视觉场景演变，所得的世界模型捕获了超越外观级视频生成的可重用交互动力学。具体地，我们在具有真实动作标注的大规模机器人操作数据上预训练了一个多视图交互基础扩散世界模型A2World。我们从两个互补的角度验证学习到的动力学先验。首先，我们将A2World适应为任务或场景专用的真实世界模拟器A2World-sim，其长程滚动通过用世界模型滚动替代真实机器人滚动，支持基于模拟器的策略评估和可扩展的假设分析。其次，从相同的预训练权重出发，我们将A2World适应为视频-动作联合预测模型A2World-policy，该模型在视觉和指令条件下预测动作。在模拟基准和真实机器人环境中的实验表明，动作条件世界模型预训练产生了可迁移的动力学先验，这有益于以模拟器为中心和以策略为中心的机器人学习。

## Abstract
We study action-conditioned world modeling as a scalable way to learn transferable dynamics priors for robot learning. By pretraining a model to predict how actions drive visual scene evolution, the resulting world model captures reusable interaction dynamics beyond appearance-level video generation. Concretely, we pretrain a multi-view interactive base diffusion world model, A2World, on large-scale robot manipulation data with real action annotations. We validate the learned dynamics priors from two complementary perspectives. First, we adapt A2World into a task- or scene-specialized real-world simulator, A2World-sim, whose long-horizon rollouts support simulator-based policy evaluation and scalable what-if analysis by replacing real-robot rollouts with world model rollouts. Second, starting from the same pretrained weights, we adapt A2World into a video-action joint prediction model, A2World-policy, that predicts actions under visual and instruction conditioning. Experiments across simulation benchmarks and real-robot settings demonstrate that action-conditioned world model pretraining yields transferable dynamics priors that benefit both simulator-centric and policy-centric robot learning.

## 2. 论文提出的方法论：核心思想、关键技术细节
### 核心思想
- **动作条件世界模型预训练**：学习从初始帧和未来动作块到未来多视角视频的条件分布 \( p(o_{t+1:t+k} | o_t, a_{t+1:t+k}) \)。
- **先验迁移**：将预训练的A2World权重作为可复用先验，通过轻微修改适应为两种变体：
  - **A2World‑sim**：长程自回归模拟器，用于策略评估。
  - **A2World‑policy**：视频‑动作联合预测模型，作为指令条件机器人策略。

### 关键技术细节
| 模块 | 关键设计 |
|------|----------|
| **A2World（基础世界模型）** | 基于DiT（Diffusion Transformer），从Cosmos‑Predict2‑2B初始化，总参数量2.5B。动作通过MLP编码后，加到扩散时间步嵌入中，通过自适应层归一化调制每个DiT块。多视角生成：将多视角堆叠在时间维度，引入可学习的视角嵌入和交叉视角注意力模块。训练使用EDM去噪得分匹配目标。 |
| **A2World‑sim** | 在基础模型上加入历史记忆机制：采用**位姿引导历史采样**（Pose‑guided History Sampling），根据动作弧长均匀选择关键历史帧；注入历史信息到交叉注意力（history tokens）和自注意力（KV记忆拼接）中。采用**自强迫（Self‑forcing）训练策略**，模型在训练时周期性遇到自身生成帧，提升长程稳定性。 |
| **A2World‑policy** | MoE-like视频‑动作联合扩散模型：共享自注意力模块，但保留模态特定的AdaLN和MLP分支。视频和动作潜在变量以不同噪声级别分别加噪，使用加权联合去噪目标。推理时采用模态级分类器自由引导。初始化：从A2World加载权重，交叉注意力用Cosmos‑Predict2初始化，动作模块复制视频分支参数。总参数量3.0B。 |

### 算法/公式（文字说明）
- **A2World训练目标**：\( \mathbb{E}_{z,n}[\| \text{A2World}(z+n; \sigma, c=0, a) - z \|_2^2] \)
- **A2World‑policy训练目标**：\( \mathbb{E}_{z_v,z_a,n_v,n_a}[ w(\sigma_v)\|\hat{z}_v - z_v\|_2^2 + \lambda_a w(\sigma_a)\|\hat{z}_a - z_a\|_2^2 ] \)，其中 \(\lambda_a = 1\)，\(\sigma_v = m_v\sigma_{\text{base}}, \sigma_a = m_a\sigma_{\text{base}}\)。

1. **动作条件世界模型预训练提供可迁移的动力学先验**：相比文本条件预训练（T‑pre）或通用视频模型（C‑init），动作‑到‑视频预训练在模拟器和策略两个下游任务中均带来显著提升。

5. **位姿引导历史采样优于滑动窗口和无历史基准**：在长程滚动中保持对象持久性和动态一致性。

## 8. 不足与局限
- **动作表示简化**：统一为双臂7维位姿+夹爪，对于灵巧手或自由关节机器人可能信息损失。
- **数据集偏差**：预训练数据多为桌面操作任务，对移动操作、多智能体协作等场景覆盖不足。
- **长程稳定性**：虽然自强迫训练改善长程滚动，但200帧以上仍可能出现漂移，附录中未与更长帧实验对比。
- **真实机器人任务有限**：仅5个任务，且任务复杂度中等（无动态物体、无环境交互变化）。
- **计算成本高**：预训练需64×H200，微调需32×H200，资源门槛较高。
- **零样本泛化未探索**：论文仅在微调后评估，未测试直接零样本迁移到新任务/新机器人。
- **评价指标局限**：真实机器人成功率评估依赖人工判断，可能引入主观性；OOD实验仅覆盖LIBERO‑Plus‑Spatial这一基准。
