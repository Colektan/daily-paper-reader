## 摘要
在现实世界中实现鲁棒的机器人操作不仅需要理解当前观测，还需要记忆和动力学建模。世界动作模型（WAMs）通过联合建模视觉预见和基于当前及历史观测的动作来具备这些能力，使其成为机器人操作的一种有前景的范式。然而，现有的WAMs面临一个基本权衡：具有高效推理的方法通常仅基于有限窗口的近期观测进行条件化，因此在非马尔可夫环境中表现不佳；而保留长历史的方法则随着序列长度增加而大幅增加时间和空间成本。为解决这一挑战，我们引入了MemoryWAM，一种具有高效持久记忆的世界动作模型。MemoryWAM采用混合记忆设计，结合了近期帧、事件边界锚定帧以及总结长程历史的紧凑要点标记。一种定制的注意力机制能够检索详细的短期上下文和压缩的长期上下文，支持依赖记忆的决策，同时减少推理延迟和GPU内存使用。在模拟和现实世界的长时域、依赖记忆的操作任务中，MemoryWAM在保持有利计算效率的同时，优于强视觉-语言-动作（VLA）和WAM基线。

## Abstract
Robust robotic manipulation in the real world requires not only an understanding of the current observation, but also memory and dynamics modeling. World action models (WAMs) possess these capabilities by jointly modeling visual foresight and actions conditioned on both current and historical observations, making them a promising paradigm for robotic manipulation. However, existing WAMs face a fundamental trade-off: methods with efficient inference typically condition only on a bounded window of recent observations and therefore struggle in non-Markovian environments, whereas methods that preserve long histories incur time and space costs that grow substantially with sequence length. To address this challenge, we introduce MemoryWAM, a world action model with efficient persistent memory. MemoryWAM uses a hybrid memory design that combines recent frames, event-boundary anchor frames, and compact gist tokens that summarize long-range history. A tailored attention mechanism enables retrieval of both detailed short-term context and compressed long-term context, supporting memory-dependent decision-making with reduced inference latency and GPU memory usage. Across long-horizon, memory-dependent manipulation tasks in both simulation and the real world, MemoryWAM outperforms strong vision-language-action (VLA) and WAM baselines while maintaining favorable computational efficiency.

## 1. 核心问题与整体含义（研究动机和背景）

**核心问题**：  
现有世界动作模型（WAMs）在非马尔可夫（non-Markovian）环境中面临根本性的**记忆-效率权衡**：
- **滑动窗口方法**（如 FastWAM、Cosmos Policy）推理效率高，但只能依赖最近有限帧，丢失长程上下文，在需要回忆历史信息的任务中表现差。
- **全历史 KV 缓存方法**（如 LingBot-VA、DreamZero）保留全部历史帧，但推理延迟和 GPU 内存随序列长度线性增加，难以用于长时域任务。

**整体含义**：  
要使 WAM 在复杂、长期、依赖记忆的机器人操作中实用，必须设计一种既能持久记忆历史信息、又不会带来高昂计算开销的机制。

## 2. 论文提出的方法论

**核心思想**：  
借鉴人类认知中的**混合记忆系统**（短期记忆 + 事件边界记忆 + 长期摘要记忆），设计一种**高效持久记忆**机制，只保留少量关键信息即可维持长程上下文。

**关键技术细节**：

1. **混合记忆（Hybrid Memory）** 包含三个部分：
   - **短期记忆（Sliding Window）**：保存最近 \( N_{\text{recent}}=4 \) 帧的完整视觉 token，用于即时闭环控制。
   - **事件边界记忆（Anchor Frames）**：保存任务开始时 \( N_{\text{init}}=2 \) 帧的完整视觉 token，因为初始场景常含指令关键信息且可能被遮挡。
   - **摘要记忆（Gist Tokens）**：为每一帧添加 \( M=8 \) 个可学习摘要 token（压缩率 \( d=L/M=15 \)，每帧视觉 token 数 \( L=120 \)）。非锚定、非最近的帧被 evict，仅保留其 gist token 的 KV cache，从而将长程历史存储从 \( O(NL) \) 降为 \( O(NM) \)。

2. **注意力机制**：  
   动作 DiT 在去噪动作 token 时，同时 attends 到短期记忆、锚帧和 gist 记忆的 KV cache，实现统一上下文的检索。

3. **架构**：  
   基于预训练 Wan2.2-TI2V-5B 的视频 DiT（~5B）和镜像设计的动作 DiT（~1B，隐藏维度 1024），总参数量约 6B。  
   训练时使用视频预测作为动力学监督，推理时跳过视频生成，仅前向视频 DiT 一次更新 KV cache。

4. **公式/算法流程**（文字说明）：
   - 视频 DiT: \( C_t^v = \Phi_v(z_t, l; C_{<t}) \)  
   - 动作 DiT: \( a_{t:t+h-1} = \Phi_a(x_a^\tau, l; C_{\le t}^v) \)
   - 混合记忆缓存：\( C_{\le t}^v = C_{\text{short}}^v \cup C_{\text{anchor}}^v \cup C_{\text{gist}}^v \)
