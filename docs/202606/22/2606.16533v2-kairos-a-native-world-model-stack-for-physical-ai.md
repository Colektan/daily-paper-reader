## 摘要
世界模型正从被动的视觉生成器转变为物理AI的基础性、操作性基础设施：它们必须从异质经验中原生地获取世界知识，在长时间跨度上维持持久状态，并在实际部署约束下高效执行。我们提出Kairos，一个围绕这些需求设计的原生世界模型栈。（1）Kairos通过学习世界，开创了由跨具身数据课程主导的原生预训练范式，该范式将开放世界视频、人类行为数据和机器人交互组织成一条渐进式发展路径。（2）Kairos通过统一的世界理解、生成和预测，在配备混合线性时间注意力的原生统一架构中维持世界，其中滑动窗口注意力捕捉局部动态，扩张滑动窗口捕捉中程依赖，门控线性注意力维持持久全局记忆。我们建立了形式化理论边界，证明这种时间分解严格限制了误差累积，在数学上保证了跨扩展时间跨度的状态传播。（3）Kairos通过纳入部署感知的系统协同设计来运行世界，支持在服务器和消费级硬件上为真实世界的观察-行动-反馈循环生成低延迟的展开。在具身世界模型、长时域和动作策略基准上的实验表明，Kairos在实现顶级性能的同时，提供了强大的效率-能力权衡。这些结果共同将Kairos定位为未来自我进化物理智能的连贯操作基础。

## Abstract
World models are transitioning from passive visual generators to foundational, operational infrastructure for Physical AI: they must natively acquire world knowledge from heterogeneous experience, maintain persistent states over long horizons, and execute efficiently within real deployment constraints. We introduce Kairos, a native world model stack designed around these requirements. (1) Kairos learns the world by pioneering a Native Pre-training Paradigm governed by a Cross-Embodiment Data Curriculum, which organizes open-world videos, human behavioral data, and robot interactions into a progressive developmental pathway. (2) Kairos maintains the world by unified world understanding, generation, and prediction within a Native Unified Architecture equipped with Hybrid Linear Temporal Attention, where sliding-window attention captures local dynamics, dilated sliding windows capture mid-range dependencies, and gated linear attention maintains persistent global memory. We establish formal theoretical bounds demonstrating that this temporal factorization strictly limits error accumulation, mathematically guaranteeing state propagation across extended horizons. (3) Kairos runs the world by incorporating a Deployment-Aware System Co-Design to support low-latency rollout generation on server and consumer-grade hardware for real-world observation-action-feedback loops. Experiments on embodied world-model, long-horizon, and action-policy benchmarks show that Kairos achieves top level performance while offering a strong efficiency-capability trade-off. Together, these results position Kairos as a cohesive operational foundation for future self-evolving physical intelligence.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
Kairos提出三层原生设计：
- **Learn the world**：通过跨具身数据课程（Cross-Embodiment Data Curriculum, CEDC）实现原生预训练，从被动观察进化到主动控制。
- **Maintain the world**：原生统一架构（理解+生成+预测）搭配混合线性时间注意力，理论保证长时域状态保持。
- **Run the world**：部署感知的系统协同设计，实现低延迟、低内存的实时推理。

### 关键技术细节

1. **跨具身数据课程（CEDC）**：
   - **阶段I（物理预训练）**：利用百万小时开放世界视频，学习通用物理规律（重力、碰撞、流体等）。采用渐进式训练：图像预训练（256p）→ 图像-视频混合预训练（逐步提升至720p）→ 持续预训练（最大241帧）。使用形状感知指数时间步移位调度器。
   - **阶段II（人类中心行为预训练）**：利用10万小时人类演示（第一/第三人称），学习任务结构行为语义。四个子阶段：人类中心预训练→机器人中心训练→目标具身微调→RL（DPO）对齐。
   - **阶段III（联合世界-动作训练）**：训练ActionDiT，与预训练VideoDiT协同优化。使用混合注意力掩码：历史视频、未来视频、未来动作令牌分组，动作令牌采用全注意力，视频令牌采用稀疏时空注意力。损失函数：L = L_video + λ L_action。

2. **统一理解-生成-预测架构**：
   - **理解模块**：基于VLM（Qwen系列）提取高层语义表征。
   - **生成模块**：基于Diffusion Transformer（DiT）进行条件视频生成，支持T2V、I2V、TI2V。核心创新是混合线性时间注意力。
   - **预测模块**：世界动作模型（WAM），将机器人动作视为未来世界演化的一部分，联合建模视频和动作。

3. **混合线性时间注意力（Hybrid Linear Temporal Attention）**：
   - **门控线性注意力（GLA）**：基于GatedDeltaNet，实现线性复杂度的全局因果记忆。通过门控delta更新规则，具有收缩性（ρ<1），保证误差不累积。
   - **滑动窗口注意力（SWA）**：捕获局部动态，窗口大小为相邻若干帧。
   - **扩张滑动窗口注意力（DSWA）**：引入扩张因子捕捉中程依赖（如1秒量级）。
   - 理论证明（Theorem 4）：混合多尺度记忆近似充分，长时域超额风险受限于分支近似误差和几何衰减的全局记忆扰动项。

4. **部署感知系统协同设计**：
   - **时间步蒸馏**：使用分布匹配蒸馏（DMD）+一致性蒸馏（CM）将教师模型压缩为4步生成器。
   - **硬件感知推理优化**：混合并行策略（序列并行+张量并行）、DiT-Cache、编译与内核融合、FP8低精度计算、权重INT4量化、瓦片化门控DeltaNet等。在A800上可达到480P实时生成，比Cosmos-14B快28-85倍。

  - 数据收集存在自然偏差（互联网、特定机器人平台），可能影响泛化到极端场景。
  - 自动评估（如VQA）可能忽视物理细节错误，人工评估样本量较小。
- **应用限制**：
  - 模型规模较大（4B参数），虽然比14B小但仍有部署门槛（需23.5GB显存）。
  - 自我进化部分仅提出概念和初始验证（提示对齐），未大规模实验。
  - 训练数据量和计算成本高，复现门槛高。
- **方法细节**：
  - 混合注意力设计虽然理论好，但工程实现复杂，依赖定制算子。
  - 蒸馏过程需要教师模型，可能引入额外训练开销。
