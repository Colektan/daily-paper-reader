## 摘要
我们提出SLS^2，一个利用学习到的潜在世界模型中的鲁棒模型预测控制(MPC)实现从像素进行安全反馈运动规划的框架。我们的方法训练一个动作条件联合嵌入世界模型，具有紧凑的马尔可夫潜在状态，从而能够通过学习的潜在动力学进行高效的基于梯度的轨迹优化。为了在潜在预测不完美的情况下确保真实系统的安全性，我们采用共形预测对GPU加速的系统级综合(SLS)鲁棒MPC方案进行校准，获得校准后的潜在误差界和鲁棒潜在空间约束集。我们还学习并共形化一个潜在约束检查器，使得SLS规划器能够在闭环执行期间施加概率安全约束。我们在基于视觉的控制任务上评估了该方法，与潜在的预世界模型和安全规划基线相比，该方法在目标到达性能和安全性方面均有提升。

## Abstract
We present SLS^2, a framework for safe feedback motion planning from pixels using robust model predictive control (MPC) in learned latent world models. Our approach trains an action-conditioned joint-embedding world model with compact Markovian latent states, enabling efficient gradient-based trajectory optimization through learned latent dynamics. To enforce safety for the true system despite imperfect latent predictions, we inform a GPU-accelerated system level synthesis (SLS) robust MPC scheme with conformal prediction to obtain calibrated latent error bounds and robust latent-space constraint sets. We further learn and conformalize a latent constraint checker, allowing the SLS planner to impose probabilistic safety constraints during closed-loop execution. We evaluate our method on vision-based control tasks, where it improves both goal-reaching performance and safety over latent world-model and safe-planning baselines.

## 2. 方法论

### 2.1 核心思想
- 学习一个紧凑的马尔可夫潜在世界模型，使其具备梯度可微性以支持高效轨迹优化。
- 利用共形预测（Conformal Prediction，CP）对潜在动力学预测误差和潜在状态分布外检测进行校准，得到统计上有保证的不确定性集。
- 将校准后的不确定性集嵌入到系统级综合（System Level Synthesis，SLS）框架中，构建鲁棒MPC，实现可计算的、概率安全的可达管约束规划。

### 2.2 关键技术细节
1. **潜在世界模型训练**：
   - 采用LeWorldModel风格的联合嵌入预测架构（JEPA），编码器为ViT-Tiny，输出紧凑的潜在嵌入 \(z_t\)。
   - 通过历史嵌入构造马尔可夫潜在状态 \(s_t = [z_t, \Delta z_t, \dots, \Delta^K z_t]\)。
   - 使用MLP作为动力学预测器 \(f_{\phi}\)，通过多步自回归预测损失、SIGReg正则化和时间拉直损失进行训练。

2. **不确定性量化与共形校准**：
   - **潜在动力学误差**：在保留的校准集上计算残差，学习协方差模型 \(\Sigma_{\psi}(s,u)\)，并用共形预测校准缩放因子 \(q\)，得到椭圆误差集 \(E(s,u) = \sqrt{q} L_{\psi}(s,u) \mathcal{B}^{n_s}\)。
   - **分布内支持集**：利用校准潜在状态的马氏距离拟合高斯椭球，并通过共形预测确定阈值，得到在域集 \(\mathcal{I}\)。

3. **鲁棒MPC（SLS）**：
   - 在名义轨迹 \((\zeta, v)\) 附近线性化动力学，得到 \(A_k, B_k\)。
   - 将共形化误差集作为SLS的扰动注入项 \(E_k = \sqrt{q} L_{\psi}(\zeta_k, v_k)\)。
   - SLS优化响应矩阵 \(\Phi^s, \Phi^u\)，构建状态和控制可达管 \(R^s_k, R^u_k\)，并通过约束收紧 \(\tau_{i,k}\) 确保鲁棒约束满足。
   - 最终优化问题包括名义成本、管大小正则化、SLS动态约束、共形化安全约束和域内约束。

4. **潜在安全分类器**：
   - 针对每个任务训练一个潜在空间分类器 \(c_{\psi}\)，输出安全得分。
   - 利用违反约束的校准样本进行共形校准，得到保守的阈值 \(\eta\)。
   - 规划时要求 \(c_{\psi}(\Pi_z s) \geq \eta\)，转化为约束 \(g_{\psi}(s) \leq -\eta\)。

## 6. 主要结论与发现

- 名义规划方面：提出的马尔可夫潜在世界模型+梯度规划器在四个任务上均显著优于现有世界模型规划基线，尤其在Push-T和Rope任务上优势明显。

## 8. 不足与局限
