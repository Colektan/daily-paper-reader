## 摘要
世界模型正在从被动的视觉生成器转变为物理人工智能的基础性、可操作的基础设施：它们必须原生地从异构经验中获取世界知识，在长时间跨度上保持持久状态，并在实际部署约束下高效执行。我们提出了Kairos，一个围绕这些需求设计的原生世界模型栈。(1) Kairos通过开创性地采用由跨具身数据课程主导的原生预训练范式来学习世界，该课程将开放世界视频、人类行为数据和机器人交互组织成渐进式发展路径。(2) Kairos通过在配备混合线性时序注意力的原生统一架构中统一世界理解、生成和预测来维持世界，其中滑动窗口注意力捕获局部动态，空洞滑动窗口捕获中程依赖，门控线性注意力保持持久全局记忆。我们建立了形式化理论界限，证明这种时序分解严格限制了误差累积，从数学上保证了跨扩展时间范围的状态传播。(3) Kairos通过整合部署感知的系统协同设计来运行世界，支持在服务器和消费级硬件上为真实世界的观察-行动-反馈循环生成低延迟的展开。在具身世界模型、长时域和行动策略基准上的实验表明，Kairos在提供强效率-能力权衡的同时达到了顶级性能。这些结果共同将Kairos定位为未来自我进化的物理智能的一致操作基础。

## Abstract
World models are transitioning from passive visual generators to foundational, operational infrastructure for Physical AI: they must natively acquire world knowledge from heterogeneous experience, maintain persistent states over long horizons, and execute efficiently within real deployment constraints. We introduce Kairos, a native world model stack designed around these requirements. (1) Kairos learns the world by pioneering a Native Pre-training Paradigm governed by a Cross-Embodiment Data Curriculum, which organizes open-world videos, human behavioral data, and robot interactions into a progressive developmental pathway. (2) Kairos maintains the world by unified world understanding, generation, and prediction within a Native Unified Architecture equipped with Hybrid Linear Temporal Attention, where sliding-window attention captures local dynamics, dilated sliding windows capture mid-range dependencies, and gated linear attention maintains persistent global memory. We establish formal theoretical bounds demonstrating that this temporal factorization strictly limits error accumulation, mathematically guaranteeing state propagation across extended horizons. (3) Kairos runs the world by incorporating a Deployment-Aware System Co-Design to support low-latency rollout generation on server and consumer-grade hardware for real-world observation-action-feedback loops. Experiments on embodied world-model, long-horizon, and action-policy benchmarks show that Kairos achieves top level performance while offering a strong efficiency-capability trade-off. Together, these results position Kairos as a cohesive operational foundation for future self-evolving physical intelligence.

  - **部署与闭环运行困难**：推理延迟、内存占用过高，使模型无法进入实时的观察-行动-反馈循环，阻碍自我进化学习。
- **整体含义**：Kairos 提出一个原生世界模型栈，从架构层面统一“如何学习世界”、“如何维持世界状态”、“如何运行世界”三个问题，为未来自进化物理智能奠定操作基础。

## 2. 方法论

### 核心思想
- **原生预训练范式**：拒绝后训练/微调路线，而是通过**跨具身数据课程（Cross-Embodiment Data Curriculum, CEDC）** ，从模型规模初始阶段就内生注入物理规律、行为语义和具身接地。
- **统一理解-生成-预测架构**：在世界理解（VLM）、世界生成（扩散Transformer）和世界预测（动作DiT）三个模块间共享统一的世界状态表示，通过**混合线性时序注意力**维持长期一致性。
- **部署感知协同设计**：将推断效率作为一阶建模原则，通过时间步蒸馏、硬件感知优化等方法实现消费级硬件上的实时闭环推理。

### 关键技术细节
1. **混合线性时序注意力（Hybrid Linear Temporal Attention）**：
   - **滑动窗口注意力（SWA）**：捕获局部动态，窗口大小为若干帧。
   - **空洞滑动窗口注意力（DSWA）**：引入膨胀因子捕获中程依赖（约1秒量级）。
   - **门控线性注意力（GLA，基于GatedDeltaNet）**：作为全局因果记忆，利用δ更新规则和门控机制实现线性复杂度下的长期状态传播。
   - 理论分析（第2.3节及附录B）证明：纯局部注意力必然产生不可约误差（定理1），而该混合结构在近似贝叶斯预测时误差有界（定理2），门控机制确保误差不无限积累。

2. **跨具身数据课程（CEDC）**：
   - 阶段I：物理预训练——大规模开放世界视频（数亿片段），学习物理规律（重力、碰撞等）。
   - 阶段II：人类中心行为预训练——超10万小时人类操作视频，提取任务结构和高层行为语义。
   - 阶段III：联合世界-行动训练——融合机器人轨迹（AgiBotWorld-Beta, Droid等），同时优化视频DiT和动作DiT。
   - 训练目标：Flow Matching，含形状自适应时间步调度。

3. **部署感知优化**：
   - **时间步蒸馏**：将480P具身模型的采样步数压缩至4步（分布匹配蒸馏+一致性蒸馏混合目标）。
   - **硬件感知推理**：混合并行策略（序列并行+张量并行）、DiT-Cache、FP8低精度、INT4文本编码器量化、Tile-Based流式访问等。

### 公式与算法（文字说明）
- 训练损失：`L = L_video + λ L_action`，两者均为流匹配MSE。
- GLA更新：`S_t = α_t S_{t-1} + β_t (v_t - v_old_t) k_t^T`，其中α为遗忘门，β为写强度，实现了在线回归的δ规则。
- 蒸馏目标：`L = L_CM + λ_score L_DMD`，同时对齐分布和ODE轨迹。

## 3. 实验设计

### 数据集/场景
- **世界模型基准**：WorldModelBench（机器人子集+全集）、DreamGen Bench、PAI-Bench（含机器人子集和全部7个领域）、VideoPhy。
- **行动模型基准**：LIBERO-Plus（环境扰动泛化）、RoboTwin 2.0（双臂操作）。
- **长时生成**：PAI-Bench 15秒版本。

### 对比方法
- **世界模型**：Cosmos-Predict2.5 (2B/14B)、Cosmos3-Nano (16B)、Wan2.2 (5B/14B)、Lingbot (28B)、ABot-Physworld (14B)、GigaWorld-0 (2B) 等。
- **行动模型**：VLA类（π0.5, starVLA, ABot-M0, G0.5等）、WAM类（MotuBrain, SANTS, Being-H0.7, Fast-WAM等）。

### 主要结果
- **WorldModelBench-robot**：Kairos-4B总分9.30，SOTA，与Cosmos3-Nano并列指令跟随最佳，物理遵循和常识得分最高。
- **DreamGen Bench**：平均物理遵循0.538，平均总分0.618，均为SOTA。
- **PAI-Bench-robot**：领域得分88.59，总分82.57，在小模型（<10B）中第一，超越部分14B/16B模型。
- **LIBERO-Plus**：Kairos-joint平均89.0（联合训练）→90.8（联合去噪），SOTA。
- **RoboTwin 2.0**：平均96.1，SOTA（第二为MotuBrain 96.0）。
- **长时生成（15秒）** ：Kairos总分79.9，SOTA，且退化幅度小于对比模型。
- **人类评估**：在PAI-Bench、WorldModelBench、DreamGen上对Cosmos-Predict2.5、Wan2.2、Lingbot等取得显著胜率。

## 6. 主要结论与发现

- Kairos在具身世界模型和世界行动模型两类基准上均达到SOTA，且参数效率极高（4B模型超越14B-28B模型）。
- 线性混合注意力机制在保证长期一致性的同时实现了推理时线性扩展，显著提升长视频生成性能。
- 原生预训练范式（CEDC）有效弥合了异构数据鸿沟，人类中心数据提供了关键的行为先验。
- 生成-预测联合训练和联合去噪均能提升动作预测的鲁棒性和准确性。
- 理论分析证明了纯局部注意力的不可约误差，并验证了混合记忆架构的近似充分性。
