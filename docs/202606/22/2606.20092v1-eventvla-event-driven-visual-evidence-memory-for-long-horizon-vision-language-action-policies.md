## 摘要
记忆仍然是长周期机器人操作的关键瓶颈，因为标准的视觉-语言-行动（VLA）策略在任务相关线索随时间变得遮挡或不可观察时往往会失败。虽然现有的记忆增强方法利用历史上下文，但它们要么遭受严重的信息瓶颈，通过解耦的双系统引入高延迟，要么依赖于积累大量视觉冗余的无选择性缓冲区。为了解决这些限制，我们引入了EventVLA，这是一个端到端框架，基于稀疏视觉证据记忆的概念，包含两个核心组件：用于保留初始和短期上下文的视觉锚点，以及一个动态关键帧证据记忆（KEM）模块。具体来说，KEM直接从VLA的潜在嵌入中预测未来关键帧概率，以自主捕获和存储稀疏的、任务关键性的视觉事件。这种前瞻性驱动机制使策略能够动态评估当前观察的未来因果效用，在瞬态视觉证据变得不可观察之前保留它。此外，我们提出了RoboTwin-MeM，一个专门设计用于评估具有交互式视觉证据的非马尔可夫操作任务的诊断基准。大量评估表明，在17个需要记忆的仿真任务和4个真实世界双手操作任务中，EventVLA相较于最先进的记忆增强VLA实现了平均成功率提升+40%。

## Abstract
Memory remains a critical bottleneck for long-horizon robotic manipulation, as standard Vision-Language-Action (VLA) policies often fail when task-relevant cues become occluded or unobservable over time. While existing memory-augmented methods utilize historical context, they either suffer from severe information bottlenecks, incur high latency via decoupled dual systems, or rely on unselective buffers that accumulate massive visual redundancies. To address these limitations, we introduce EventVLA, an end-to-end framework founded on the concept of sparse visual evidence memory that comprises two core components: foundational visual anchors to retain initial and short-term contexts, and a dynamic Keyframe Evidence Memory (KEM) module. Specifically, KEM directly predicts future keyframe probabilities from the VLA's latent embeddings to autonomously capture and store sparse, task-critical visual events. This foresight-driven mechanism empowers the policy to dynamically evaluate the future causal utility of current observations, preserving transient visual evidence before it becomes unobservable. Furthermore, we propose RoboTwin-MeM, a diagnostic benchmark specifically designed to evaluate non-Markovian manipulation tasks with interactive visual evidence. Extensive evaluations show that across 17 memory-requiring simulation tasks and 4 real-world bimanual tasks, EventVLA achieves an average success rate improvement of +40% over state-of-the-art memory-augmented VLAs.

- **动机**：需要一种既能保留关键瞬态线索、又能避免冗余的稀疏视觉证据记忆机制。

## 2. 方法论：核心思想、关键技术细节

### 2.1 核心思想
- **稀疏视觉证据记忆**：由两部分组成
  - **基础视觉锚点**（Visual Anchors）：初始帧 `o0`（永久空间参考）+ 短期历史滑动窗口 `{o_{t-K}, ..., o_{t-1}}`（局部运动线索）。
  - **动态关键帧证据记忆（KEM）**：前瞻性地从VLA隐状态预测未来时间步是否为关键帧，自主决定何时将当前观察写入事件缓冲区。
- **整体流程**：动作生成 `a_t = π(o_t, M_{t-1}, l)`，其中 `M_t = A_t ∪ E_t`。

### 2.2 关键技术细节
- **KEM模块**：轻量级MLP头，与动作头并行，输入VLA最终层隐状态 `h_t ∈ R^{H×d}`，输出未来H步的关键帧概率向量 `p̂_t = σ(MLP(h_t)) ∈ [0,1]^H`。
- **写入触发**：当 `p̂_t[i] ≥ τ_commit`（阈值0.55）时，将第i步的原始图像存入事件缓冲区 `E_t`，缓冲区容量 `N_max=5`，FIFO淘汰。
- **训练**：

  - 软标签：用升余弦核在事件周围R半径内生成连续标签，避免二值硬标签的噪波。
  - 损失：`L = L_action + λ L_BCE`，其中L_BCE基于软标签。

## 6. 主要结论与发现
- **锚点有效性**：基础视觉锚点（初始帧+短期历史）在RMBench上达到67.8%平均成功率，远优于所有非记忆模型（<11%）和大部分记忆模型（最佳42.0%）。

- **消融关键**：隐式记忆体、硬标签、无NMS、小缓冲区、短动作块均导致性能大幅下降，证明显式图像拼接、软监督、稀疏后处理、足够预测窗口不可或缺。
- **推理速度**：KEM模块仅增加少量延迟（约1.09秒/块 vs 基线0.36秒），满足实时控制需求（0.94Hz）。

## 7. 优点
1. **创新记忆范式**：抛弃了全历史缓存或压缩隐状态，采用**稀疏前瞻性关键帧捕获**，平衡信息保留与计算效率。
2. **端到端可训练**：KEM作为轻量头与动作头联合训练，无需额外规划模块，避免了双系统误差传播。
3. **自动标注管线**：利用VLM自动提取关键帧，降低了人工成本，且通过软标签容忍时间偏差。
4. **诊断性Benchmark**：RoboTwin-MeM明确参数化中间事件数量，能够细粒度评测记忆能力，填补了现有基准的空白。
5. **充分泛化验证**：仿真→真实、双臂→单臂、马尔可夫→非马尔可夫、多种基线全覆盖。

## 8. 不足与局限
1. **事件密度上限**：固定容量缓冲区（N_max=5）在超高密度长任务（>10分钟，事件数>5）中可能溢出，导致早期关键证据被淘汰。
2. **复杂度扩展性**：论文未测试更复杂场景（如连续数百个事件），也未探讨层级记忆或压缩代替方案。
3. **真实任务规模**：仅4个真实任务，每个20次试验，样本量较小，可能存在随机波动。
4. **自动化标注偏差**：Qwen3-VL标注有<10步（仿真）或<50步（真实）误差，尽管软标签缓解，但极端情况下可能错误定位事件。
5. **计算开销**：推理时需要处理多个历史帧（锚点+最多5个关键帧），内存占用略高于纯反应模型，但文中已给出吞吐率，表明可接受。
6. **缺乏长期RL训练**：该方法仅基于行为克隆（OFT），未探索从失败中强化学习以改进记忆策略。
