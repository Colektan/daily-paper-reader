## 摘要
机器人操作的强化学习通常需要手动设计密集的奖励函数，这难以调优且脆弱，或者从人类示教或偏好中学习奖励，这成本高昂。最近一系列工作使用预训练的视觉语言模型作为零样本奖励模型，用单个文本提示替代这些成本。然而，我们认为单一的全局提示对于初始条件随机化的长时程操作任务过于粗糙。单提示的VLM奖励在大部分轨迹上接近平坦，使得智能体难以检测早期进展。我们提出强化微任务学习（RMTL），该方法将操作任务分解为少量由语言描述的微任务，并训练智能体在它们之间切换。每一步，智能体收到使用当前活跃微任务的提示计算的多视角VLM奖励，并跨多个相机视角平均以减少视角特定遮挡的影响。反向课程逐渐将智能体暴露于更困难的初始条件，而PPO工人首先使用固定的基于距离的规则训练，该规则选择活跃微任务。然后我们用学习的层次管理器替换该规则，将基于规则的阶段选择转变为完全学习的层次策略。我们在Fetch操作环境中实例化RMTL，使用三个短阶段特定提示，无需额外的提示调优。实验表明，RMTL比单提示VLM奖励提供更具信息的奖励信号，从而实现更快的学习。这些结果表明，将VLM奖励分解为微任务特定的语言提示可以显著提高语言引导强化学习在机器人操作中的可扩展性。

## Abstract
Reinforcement learning (RL) for robotic manipulation often requires manually designing a dense reward function, which is difficult to tune and often fragile, or learning a reward from human demonstrations or preferences, which can be expensive. A recent line of work uses pretrained vision-language models (VLMs) as zero-shot reward models, replacing these costs with a single text prompt. However, we argue that a single global prompt is too coarse for long-horizon manipulation tasks with randomized initial conditions. The single-prompt VLM reward is near-flat for much of the trajectory, making early progress hard for the agent to detect. We propose Reinforced Micro-Task Learning (RMTL), an approach that decomposes a manipulation task into a small set of language-described micro-tasks and trains the agent to switch between them. At each step, the agent receives a multi-view VLM reward computed using the prompt of the currently active micro-task and averaged across multiple camera views to reduce the effect of view-specific occlusions. A reverse curriculum gradually exposes the agent to harder initial conditions, while a PPO worker is first trained with a fixed distance-based rule that selects the active micro-task. We then replace this rule with a learned hierarchical manager, turning rule-based phase selection into a fully learned hierarchical policy. We instantiate RMTL on the Fetch manipulation environment using three short stage-specific prompts and without additional prompt tuning. Experiments show that RMTL provides more informative reward signals than single-prompt VLM rewards, enabling faster learning. These results suggest that decomposing VLM rewards into micro-task-specific language prompts can substantially improve the scalability of language-guided reinforcement learning for robotic manipulation.

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
**RMTL（Reinforced Micro-task Learning）** 将长时程操作任务分解为少量**语言描述的微任务**（微任务），每个微任务对应一个阶段特定的文本提示。智能体在微任务间切换，每步仅根据**当前活跃微任务的提示**计算多视角VLM奖励的平均值。同时结合**反向课程**逐渐增加初始条件难度，并使用**层次强化学习**：第一阶段基于固定规则切换微任务训练底层PPO工人；第二阶段用学习的层次管理器替换规则，实现完全学习的层次策略。

### 关键技术细节

#### （1）多视角VLM奖励聚合
- 使用6个固定相机视角（宽视角、正面、正面侧面、等距、侧面、近景），每帧独立计算每个视角的VLM评分，再取**算术平均**作为最终奖励。
- 平均化操作平滑了单视角的遮挡和噪声，使训练信号更稳定（图3、图8）。

#### （2）分阶段VLM奖励（Phased VLM Reward）
- 采用Rocamonde et al.提出的投影风格评分器，并适配到每个微任务。
- 定义基线提示 \(b\)（如“桌面上一个红色立方体和一个机器人夹爪”）和微任务目标提示 \(\tau_i\)（共 \(K=3\) 个微任务：approach, align, grasp）。
- 对每个微任务 \(i\)，计算方向 \(d_i = e_{\tau_i} - e_b\)（文本嵌入差），然后使用投影混合矩阵 \(P_\alpha(d_i) = \alpha \frac{d_i d_i^\top}{\|d_i\|^2} + (1-\alpha) I\)，其中 \(\alpha=0.9\) 放大奖励动态范围（附录C）。
- 每视角每图像的得分：\(r_v^{(i)} = 1 - \frac{1}{2} \| (x_v - e_{\tau_i}) P_\alpha(d_i) \|_2^2\)，即沿目标-基线方向的投影距离。
- 最终奖励 \(r_t^{(i)} = \frac{1}{V} \sum_{v=1}^V r_v^{(i)}\)。

#### （3）反向课程（Reverse Curriculum）
- 将初始夹爪位姿从“易”（物体上方附近）到“难”（全工作空间随机）分为5个难度等级，基于在线成功率自动推进。
- 每个阶段设置成功阈值和最少环境步数，确保智能体掌握当前难度后再进入更难初始条件（图6）。

#### （4）阶段一：基于规则的微任务切换 + PPO工人
- 使用确定性规则：根据夹爪与立方体距离划分三个微任务（approach: 距离 > 0.15 m；align: 0.04~0.15 m；grasp: < 0.04 m）。
- 工人采用PPO（近端策略优化），策略网络为MLP [256,256]。
- 最大步长40步，成功时给予+1终结奖励。

#### （5）阶段二：学习层次管理器
- 管理器 \(\pi_M(\phi | s)\)：小MLP [128,128] + softmax，输出3个微任务的分类概率，每 \(H=3\) 环境步决策一次。
- **行为克隆（BC）热启动**：先用规则选择器收集250轮元组 \((s, i_{rule})\)，训练管理器3500步交叉熵。
- **REINFORCE优化**：在PPO rollout结束后，基于段回报（\( \sum_{t \in seg} r_t \)）的z-score优势，优化式(6)：
  \[
  L_M = -\mathbb{E}_{\text{seg}}[\log \pi_M(\phi|s) \cdot A_{\text{seg}}] - c_H \mathcal{H}[\pi_M] + w_{KL} \text{KL}(\pi_{\text{rule}} \| \pi_M)
  \]
  包含熵奖励（\(c_H=0.01\)）和KL锚定到规则策略（\(w_{KL}=0.30\)），采用ε-贪婪探索（\(\varepsilon=0.20\)）。
- 工人加载阶段一560k步检查点，前50k环境步冻结演员网络（仅训练评论家），之后解冻并以降低的学习率继续训练。

### 算法流程（文字描述）
1. 初始化VLM编码器、PPO工人、规则选择器。
2. **阶段一**：使用规则选择微任务，工人通过PPO在反向课程中训练，直到收敛。
3. **阶段二**：
   - 用规则选择器生成数据，BC训练管理器。

## 6. 主要结论与发现

1. **微任务分解是关键**：将单全局提示分解为3个阶段特定提示，将平坦奖励信号转变为分段单调信号，使得PPO能够有效学习（图5）。
2. **多视角聚合提升稳定性**：视角均值比任何单视角更平滑，降低了遮挡噪声，且不改变梯度形状（图8）。
3. **反向课程是必要粘合剂**：没有课程，直接随机初始化时VLM奖励太不具信息化，无法引导探索（图6）。
4. **学习管理器超越规则**：学习的管理器（98%成功率）优于基于距离的规则（94%），且规则仍可作为有效的BC教师和KL锚，使得管理器快速达到并超越规则上限（图7）。
5. **层次结构有效性**：阶段二的两层层次策略（管理器+工人）实现了比非层次PPO（89%）和单提示PPO（0%）显著更高的成功率。

2. **模块化设计**：多视角聚合、反向课程、层次管理器可独立替换或改进，便于后续研究。

## 8. 不足与局限

1. **实验覆盖不足**：仅在单一环境（FetchPickAndPlace）上验证，未推广到其他操作任务（如推、旋转、多物体）或真实机器人场景，泛化性存疑。

## 9. 对未来工作的启示

- **自动提示生成**：可利用大语言模型（LLM）根据任务描述或场景自动生成微任务提示，消除手动设计阶段。  
- **跨任务泛化验证**：在更多操作基准（如Meta-World、Adroit）或真实机器人上测试RMTL，评估迁移能力。  
- **改进VLM的时间一致性**：当前VLM基于静态帧（单帧时间），可引入视频级VLM（如VideoCLIP）或时序模型，以捕捉动态接触信号。  
- **减少视角数量**：通过可学习的视角选择或注意力机制，在保持奖励精度的同时降低计算量。  
- **端到端联合训练**：将管理器与工人的训练完全同步，避免两阶段依赖，探索更稳定的联合优化方法（如POMDP中的端到端分层RL）。  

## 10. 综合评价

本文提出的RMTL方法有效解决了VLM零样本奖励在长时程操作任务中的奖励平坦问题，通过微任务分解、多视角聚合和反向课程等技巧，使PPO在仅需少量文本提示的情况下达到接近完美的成功率（98%）。实验设计清晰、消融完备，在单一环境中验证了各组件的作用。然而，其仅在一个任务上测试且高度依赖手工渲染细节，限制了结论的泛化性。总体而言，这是一篇思路新颖、工程扎实的文章，为视觉语言模型驱动的机器人自主学习开辟了可行路径，但距离实际部署仍需弥合域差距和计算效率问题。
