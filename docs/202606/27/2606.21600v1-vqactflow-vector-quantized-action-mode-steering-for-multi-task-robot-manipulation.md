## 摘要
多任务机器人操作策略从示范中学习具有挑战性，因为传统上单个网络必须从多模态示范分布中，根据语言和视觉上下文选择不同性质的动作模式。错误的模式选择意味着执行错误的任务或场景中不可行的动作。将连续动作标记化为学习到的离散码本，在表示层面分离这些模式，为多任务学习提供了结构优势。我们提出VQActFlow，一种多任务操作策略，它将动作片段标记化并通过变分流匹配生成码序列。VQActFlow在整个生成过程中保持对动作模式的显式偏好。推理时的引导作用于这种偏好以引导模式承诺。我们通过语言条件下的无分类器引导来实现这一点，该引导将策略导向指定的动作模式，并利用学习的码本评论家提供互补的可行性信号。我们在三个平台上评估VQActFlow：LIBERO模拟基准测试、执行全身抓取放置的Unitree G1人形机器人，以及执行接触密集任务的ALOHA式双臂平台。在这些基准测试中，VQActFlow优于连续和离散基线方法。

## Abstract
Multi-task robot manipulation policies are challenging to learn from demonstration because traditionally a single network must select among qualitatively different action modes from a multimodal demonstration distribution, conditioned on language and visual context. A wrong mode selection means executing the wrong task or an action infeasible in the scene. Tokenizing continuous actions into a learned discrete codebook separates these modes at the representation level, offering structural advantages for multi-task learning. We propose VQActFlow, a multi-task manipulation policy that tokenizes action chunks and generates code sequences via Variational Flow Matching. VQActFlow maintains an explicit preference over action modes throughout generation. Inference-time guidance acts on this preference to steer mode commitment. We instantiate this with classifier-free guidance over language conditioning, which steers the policy toward the instructed action mode, and a learned codebook critic that supplies a complementary feasibility signal. We evaluate VQActFlow on three platforms: the LIBERO simulation benchmarks, a Unitree G1 humanoid performing whole-body pick-and-place, and an ALOHA-style bimanual platform performing contact-rich tasks. Across these benchmarks, VQActFlow outperforms both continuous and discrete baselines.

## 1. 论文的核心问题与整体含义

- **研究动机**：多任务机器人操作策略需要从语言指令和视觉观察中，从多模态演示分布中正确选择「动作模式」（如目标物体、接触策略、双臂协调）。传统连续生成策略（扩散/流匹配）在连续动作空间中操作，无法显式表示模式选择，导致错误任务或不可行动作。
- **核心问题**：如何让策略在生成过程中持续保持对离散动作模式的显式偏好，并在推理时高效引导这种偏好，避免模式选择错误。
- **整体含义**：提出 VQActFlow，通过向量量化将连续动作离散为码本，利用变分流匹配（Variational Flow Matching, VFM）在保持连续运输的同时输出分类概率，从而在每个生成步骤都有显式的模式偏好，为推理时引导提供统一的分类接口。

## 2. 方法论

### 核心思想
- **两阶段架构**：  
  1. 预训练 VQ-VAE 将动作片段压缩为离散码本索引序列（码本大小 K=512，序列长度 L << 动作步数 H）。  
  2. 训练 VFM 策略网络，将高斯噪声运输到码本嵌入空间，但损失函数使用交叉熵（而非 L2 回归），使网络直接输出各码本（动作模式）的 logits。
- **显式模式偏好**：每个集成步骤，网络输出 logits → softmax 获得概率分布 p(l)∈R^K，表示对每个动作模式的偏好。使用后验均值计算速度场：μ₁ = Σ p_k·e_k。
- **推理时引导**：统一作用于 logits 或概率分布。
  - **无分类器引导（CFG）**：线性外推条件 logits 和无条件 logits 得到引导后 logits（公式9），再计算概率。
  - **码本评论家（Codebook Critic）**：一个轻量 Transformer，接收观察、语言、当前概率分布 p 和时间 t，输出可行性分数。通过梯度上升调整概率分布（公式11），引导强度 λ(t) 随分布熵自适应（公式12）。训练使用对比学习（InfoNCE），负例包括时间打乱、随机码替换、错误观察、错误语言等7种，强制学习多维度可行性信号。

### 关键技术细节
- **VQ-VAE**：时序编码器将动作片段压缩为 L 个连续潜在，最近邻量化，CNN 解码器重建动作，码本通过指数移动平均更新。
- **VFM 训练**：最优传输插值 xt=(1-t)x0 + t x1，交叉熵损失直接对码本索引监督，不提供速度场显式梯度。
- **推理**：Euler 积分从 t=0 到 1，最终状态最近邻量化得到码索引，解码为动作。
- **网络架构**：策略为 Diffusion Transformer (DiT)，12 层，d_model=1024，8 头；视觉编码器 ResNet-18，文本编码器冻结 CLIP-B/32；评论家为 3 层 Transformer，256 隐藏维。

## 6. 论文的主要结论与发现

1. **CFG 通过分类接口比连续接口更有效**：在 LIBERO-Goal 上，VQActFlow 在 w=4 达到 81.0%，而 Discrete Policy 在 w=6 仅 61.5%，且 VQActFlow 方差更小。
2. **码本评论家提供互补信号**：在 LIBERO-90 上，CFG 加评论家比单独 CFG 提升约 2.9%；在双臂任务上，CFG+评论家达 77.5%，优于单独 CFG 的 73.8%。
3. **离散化带来更平滑动作**：CFM 在双臂硬件上产生严重高频振荡（峰值 jerk 约 10^4 rad/s³），而 VQActFlow 仅为 2×10² rad/s³，归因于 VQ-VAE 的时序压缩和组合词汇编码。
4. **最优 CFG 权重需要平衡**：过低导致错误任务，过高导致执行精度下降（如抓偏）。
5. **在 LIBERO-90 上 VQActFlow 以 80.5% 超越所有基线**（MT-ACT 72.4%，CFM 77.2%，Discrete Policy 60.3%）。

## 7. 优点

- **显式动作模式分离**：VQ-VAE 将连续动作聚合成离散码本，码本索引本身对应相似任务，结构上有利于多任务学习。
- **推理时引导的统一分类接口**：CFG 和评论家都作用于 logits/概率，可简单叠加，无需改变生成过程。
- **评论家无需在线 rollout 或仿真**：仅用演示数据通过对比学习训练，可自监督提供可行性信号，且与策略解耦。
- **动作平滑性**：VQ-VAE 的时序压缩和码本组合自然抑制了连续策略常见的高频振荡。
- **变分流匹配创新**：在连续运输中加入交叉熵损失，保留了离散结构，同时允许连续速度场定义。
- **实验结论清晰**：通过多次消融验证了各组件的贡献，对比公平。
