## 摘要
奖励设计仍然是机器人运动强化学习中的一个主要瓶颈，成功策略往往依赖于精心调整的任务特定奖励函数。基于偏好的强化学习提供了一种替代方案，但现有的基于LLM的方法通常要求对行为之间进行单一的整体判断，这使得捕捉高质量运动背后的多个竞争目标变得困难。我们提出了多目标AI驱动的偏好学习（MAPL），这是一个从高层级自然语言目标中学习运动奖励的框架，而不是手动设计的奖励方程。MAPL提示大型语言模型沿着语义上有意义的准则独立比较轨迹，使用与地形无关且不需要领域专业知识的通用语言描述。这些目标维度的偏好用于训练多头偏好评分模型，其输出被聚合形成用于策略优化的标量奖励。在四个四足运动环境中，MAPL仅使用LLM生成的偏好训练策略，并实现了与专家设计的奖励相当或更优的性能，同时消除了特定任务的奖励工程。

## Abstract
Reward design remains a major bottleneck in reinforcement learning for robot locomotion, where successful policies often depend on carefully tuned, task-specific reward functions. Preference-based reinforcement learning offers an alternative, but existing LLM-based methods typically ask for a single overall judgment between behaviors, making it difficult to capture the multiple competing objectives that underlie high-quality locomotion. We present Multi-Objective AI-Informed Preference Learning (MAPL), a framework that learns locomotion rewards from high-level natural language objectives rather than manually engineered reward equations. MAPL prompts a large language model to compare trajectories independently along semantically meaningful criteria, using generic language descriptions that are terrain-invariant and require little domain expertise. These objective-wise preferences are used to train a multi-head preference scoring model, whose outputs are aggregated to form a scalar reward for policy optimization. Across four quadruped locomotion environments, MAPL trains policies using only LLM-generated preferences and achieves performance comparable to or better than expert-designed rewards, while eliminating task-specific reward engineering.

## 2. 论文提出的方法论
- **核心思想**：将机器人运动目标分解为三个语义明确、与地形无关的准则——**速度跟踪**（Velocity Tracking）、**稳定性**（Stability）和**平滑度**（Smoothness）。每个准则用纯自然语言定义，无需领域专业知识或手工公式。
- **关键技术细节**：
  - **多目标偏好生成**：对于每个轨迹对，LLM被提示分别输出三个维度的偏好标签（0/1/0.5），并多次查询取多数票以降低随机性。
  - **多头偏好评分模型**：使用Transformer架构，共享底层参数，每个目标有一个独立的输出头，输出每个状态下的偏好分数 \(\sigma_\psi^{(i)}(s)\)。
  - **Bradley-Terry模型**：基于偏好标签训练模型，最小化交叉熵损失：
    \[
    P_\psi(\tau^1 \succ \tau^0) = \frac{\exp\left(\sum_t \sigma_\psi(s_t^1)\right)}{\sum_{i\in\{0,1\}} \exp\left(\sum_t \sigma_\psi(s_t^i)\right)}
    \]
  - **聚合与奖励**：将各目标得分线性组合：\(\sigma_\psi(s_t) = \sum_i \lambda_i \sigma_\psi^{(i)}(s_t)\)，其中\(\lambda\)为可调权重（速度与稳定性权重较大，平滑度根据地形调整）。最终奖励采用**势差形式**：\(r(s_t) = \sigma_\psi(s_t) - \sigma_\psi(s_{t-1})\)，以提高样本效率。
  - **算法流程**（Algorithm 1）：初始化缓冲区 → 每T步更新评分模型（采样轨迹对→LLM标注→更新模型参数）→ 每步滚动策略→计算势差奖励→更新策略（PPO）。评分模型更新频率约为策略更新频率的1/100。

## 7. 优点
- **消除手工奖励工程**：仅需用自然语言描述目标，无需领域专家手工设计复杂奖励公式，极大降低了应用门槛。
- **多目标独立建模**：将机器人运动分解为语义清晰、与地形无关的三个准则，提高了LLM排名的准确性和稳定性，避免了单目标偏好中的权衡困难。
- **通用性**：同一套自然语言提示可应用于多种不同地形，无需任务特定修改。
- **势差奖励设计**：将偏好得分转换为势差形式，解决了直接使用得分作为奖励时的不稳定问题，显著提升了训练效率和最终性能。
- **高效性**：仅需500个轨迹对即可训练可靠偏好模型；LLM查询次数可低至1次；策略训练在模拟器中收敛迅速。
- **实验全面**：包含丰富的消融、敏感性、排名准确性、速度误差、姿态分布等分析，验证了各组件贡献和鲁棒性。
