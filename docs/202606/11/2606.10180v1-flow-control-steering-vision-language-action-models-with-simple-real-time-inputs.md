## 摘要
我们引入了视觉-语言-动作（VLA）模型的流控制，这是一种通过通用输入（如键盘）实时引导VLA动作的简单有效方法。该方法可直接使用，无需重新训练或微调VLA。它使得相对粗糙的用户输入能够引导VLA与用户意图对齐。VLA将这些输入转换为从训练期间学到的VLA专家动作分布中抽取的动作样本，从而生成高质量（符合动作专家分布）和高保真度（反映用户意图）的动作。我们证明了流控制具有许多理想特性：（1）流控制能准确且响应迅速地用用户输入引导机器人动作；（2）对次优用户输入具有鲁棒性；（3）使用户能够引导VLA实现显著更高的成功率和更快的任务完成速度；（4）在流控制轨迹上微调VLA可改进自主策略。综上，这些结果为用户提供了一种简单直观的方式来帮助引导VLA动作，从而提升任务性能。

## Abstract
We introduce flow control of vision-language-action (VLA) models, a simple and effective way to steer VLA actions in real-time through generic inputs, such as a keyboard. This method can be used out-of-the-box and does not require retraining or fine-tuning VLAs. It enables relatively crude user inputs to steer a VLA to align with user intent. The VLA transforms these inputs into action samples drawn from the VLA expert action distribution learned during training, so that the generated actions are high quality (conformity to the action expert distribution) and high fidelity (reflecting the user's intent). We demonstrate that flow control has many desirable properties: (1) flow control accurately and responsively steers robot actions with user inputs, (2) it is robust to suboptimal user inputs, (3) it enables users to steer VLAs to achieve significantly higher success rates and faster task completion, and (4) fine-tuning a VLA on flow control trajectories improves the autonomous policy. Together, these results provide a simple and intuitive way for users to help steer VLA actions, increasing task performance.

## 2. 方法论

### 2.1 核心思想

- VLA（如π0.5）采用**流匹配动作专家（flow matching action expert）**，它通过积分一个常微分方程（ODE）将初始噪声（x0 ~ N(0,I)）转化为动作样本 x1 ~ p_expert。
- **流控制（Flow Control）** 的关键观察：流匹配模型是**微分同胚**，初始条件的信息会完整保留到最终动作 x1 中（与扩散模型不同，扩散模型在每一步添加噪声会洗掉初始信息）。因此，可以通过**修改初始条件**来引导动作。
- 具体操作：将用户输入（如键盘方向）转换为末端执行器速度，再通过逆运动学（IK）得到关节速度 x_user，然后将该值作为 ODE 的初始条件 x0 = x_user（经过归一化），之后按原流程积分得到最终动作 x1。

### 2.2 算法流程（文字说明）

- 循环执行：
  1. 观测环境（相机、语言、机器人状态）；
  2. 获取用户键盘输入（上/下/左/右/前/后），转化为笛卡尔速度，再通过IK得到关节速度 x_user；
  3. 采样一个标准高斯噪声 x0 ~ N(0,I)；
  4. 将 x_user 替换 x0 的前 τ 个时间步（动作块长度为16，通常只替换前若干步）；
  5. 进行 K 步（默认10步）欧拉积分：x_{k+1} = x_k + Δt * vθ_t(x_k|c)；
  6. 执行最终动作 x1；
- 特点：无需对VLA进行任何训练或微调，直接即插即用。

### 2.3 与现有方法的关键区别

- 相比扩散模型的共享自治方法（如Yoneda等），无需超参数γ，也不产生额外延迟；
- 相比通过强化学习策略学习初始分布的方法（DSRL），流控制无需学习任何策略，直接设定初始条件；
- 相比语言或草图引导，流控制提供连续、低带宽但精确的实时调整。

## 6. 主要结论与发现

1. **流控制可有效引导VLA**：通过修改初始条件即可改变机器人选择，在Two-Block任务中从85%选右侧变为近乎100%选左侧。
2. **生成的动作是on-policy的**：即使在引导时，流匹配专家会将冲突的初始条件转化为与专家分布一致的动作，不影响子任务完成（如仍然成功抓取和放置）。
3. **显著提升任务性能**：
   - Marker-in-Bowl：成功率从53.3%提升至99.4%，用时减少8.3%；
   - Cup-Stacking：成功率从48%提升至87.7%，用时减少30%；
4. **鲁棒性**：对次优用户输入有较强鲁棒性，且能平滑融合用户意图。
5. **微调提升自主策略**：使用流控制收集的轨迹微调后的π0.5-FC在Cup-Stacking上达到100%成功率，用时降至22秒（自主策略为50.33秒）。
6. **缩小操作者差异**：流控制使新手操作者的表现接近专家操作用户，完成时间方差显著减小。

## 7. 优点

- **零样本使用**：无需任何训练、微调或数据标注，直接应用于已有VLA（带流匹配动作头）。
- **简单直观**：仅需键盘方向键，用户学习成本低，实时响应。
- **高质量与高保真兼顾**：用户输入通过流ODE变换后仍是专家分布内的动作，既忠实又可靠。
- **底层理论清晰**：基于流匹配的微分同胚特性，论证了初始信息可完整保留。
- **附带数据收集价值**：流控制轨迹自然落在专家分布内，可用作后续微调数据，且收集成本低。

## 8. 不足与局限

- **适用模型范围有限**：仅适用于使用流匹配动作专家的VLA（如π0系列），不适用于扩散动作专家或自回归离散动作模型的VLA。
- **难以对抗VLA的“惯性”**：当VLA已经强烈承诺一个动作模式（如已决定抓取某物体），仅修改初始条件可能不足以改变其行为；文中需扰动ODE积分多个步骤才能实现“改变主意”，但该动作可能变成OOD（分布外）。
- **精度受限于VLA本性**：在物体间距极小（0cm）时，流控制可能选错（准确率67%），提示在高度模糊场景下仍有局限。
- **任务局限性**：实验仅涉及固定桌面操作任务，未测试移动机器人或复杂环境下的泛化。
- **用户输入空间有限**：当前仅6个方向，可能对更复杂操作支持不足（如旋转、多指操作等）。
- **未分析长期交互效果**：用户可能过度依赖引导而降低自主策略的改进动机。
