## 摘要
扩散策略（DPs）已成为机器人学习中的一种表达性策略表示，通常与行为克隆（BC）等模仿学习方法一起使用。然而，虽然其成功主要局限于BC，但直接进行强化学习（RL）微调仍然具有挑战性，因为动作是通过多步去噪过程生成的。在这项工作中，我们提出了MODIP，一个用于扩散策略的离线到在线微调框架。MODIP并非直接将RL应用于DPs，而是利用世界模型（WM）来指导策略适应，并保持BC的简单性和稳定性。我们利用模型预测控制（MPC）在WM内生成高质量轨迹，并将其作为监督目标来微调DP。为了提高MPC规划的效率，MODIP使用终端状态值代替依赖于策略的状态-动作值，从而减少推理时间。此外，MODIP使用独立于策略的TD目标训练评论家，从而减少训练时间。在D4RL（MuJoCo、Kitchen）和RoboMimic任务上的实验表明，MODIP改进了超越BC的扩散策略，并且与扩散策略RL微调方法以及TD-MPC2等强模型基线相比具有竞争力或更优性能。

## Abstract
Diffusion policies (DPs) have emerged as expressive policy representations for robot learning, often used with imitation learning methods such as behavioral cloning (BC). However, while their success has largely been confined to BC, direct reinforcement learning (RL) fine-tuning remains challenging because actions are generated through a multi-step denoising process. In this work, we propose MODIP, a framework for the offline-to-online fine-tuning of DPs. Rather than directly applying RL to the DPs, MODIP leverages a world model (WM) to guide policy adaptation and keeps the simplicity and stability of BC. We utilize model predictive control (MPC) to generate high-quality trajectories within the WM, and use them as supervised targets for fine-tuning the DP. To make MPC planning efficient, MODIP uses a terminal state value instead of a policy-dependent state-action value, reducing inference time. Additionally, MODIP trains critics with policy-independent TD targets, reducing training time. Experiments on D4RL (MuJoCo, Kitchen) and RoboMimic tasks show that MODIP improves diffusion policies beyond BC, and is competitive with or outperforms diffusion policy RL fine-tuning methods and strong model-based baselines such as TD-MPC2.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：扩散策略（Diffusion Policy, DP）在机器人模仿学习（如行为克隆 BC）中表现出色，能建模多模态动作分布，但由于动作通过多步去噪过程生成，直接使用强化学习（RL）进行微调非常困难，容易不稳定且计算昂贵。
- **核心问题**：如何在保持 BC 训练简单稳定的前提下，利用 RL 的信号（奖励）有效改进预训练的 DP，使其超越演示数据的性能。
- **整体含义**：提出一种称为 MODIP 的离线到在线微调框架，通过结合世界模型、模型预测控制（MPC）和策略蒸馏，避免直接对 DP 进行 RL 优化，而是用 MPC 生成更优轨迹作为监督目标，从而间接提升 DP。

## 2. 论文提出的方法论

### 核心思想
- 使用一个学习到的潜在世界模型（latent world model），结合模型预测控制（MPC）在潜在空间中进行轨迹优化。
- 将预训练的 DP 作为 MPC 的成熟动作序列先验（proposal prior），与随机采样结合生成候选轨迹。
- 通过 MPC 得到的高质量轨迹存储到在线缓冲区，然后使用标准 BC 去噪损失（即扩散模型原始的噪声预测损失）对 DP 进行蒸馏，即 MPC-to-policy distillation。
- 为提升计算效率，采用**终端状态值 V(s)** 代替常见的终端状态-动作值 Q(s, π(s))，避免在终端状态处调用耗时的 DP 推理；同时训练**策略无关的评论家（policy-independent critic）**，其 TD 目标不依赖当前策略，从而节省 critic 更新时 DP 的调用。

### 关键技术细节
1. **潜在世界模型**：包含编码器 hη、潜在动力学 fθ、奖励模型 R̂ξ。以 TOLD 架构为基础，但终端值使用 Vψ 而非 Q。
2. **DP 引导的 MPC 规划**：MPPI 优化器，候选动作序列一部分由 DP 生成，一部分从高斯分布采样。通过滚动在潜在模型中获得预测奖励和终端 V，选择最优序列并执行第一个动作。
3. **MPC 到策略蒸馏**：DP 的训练损失保持为标准的去噪损失 LDP(ϕ) = E[‖ϵ - ϵϕ(ak, k, hη(st))‖²]，但目标动作来自 MPC 优化后的轨迹。此外加入冻结 BC 正则化（Lπ = LDP(aMPC) + λ(t)LDP(areg)），λ(t) 逐渐退火至 0。
4. **策略无关的评论家学习**：
   - TD 目标：yt = rt + γ(1 - dt) min_{j=1,2} V̄ψj(h̄η(st+1))，其中 V̄ 和 h̄ 为 EMA 目标网络。
   - 通过 AFU 风格的 value-advantage 分解训练 V 和 A，不依赖于当前策略的动作。
5. **离线到在线的混合重放**：采用类似 RLPD 的在线/离线样本混合策略，但使用时间变化的混合比例 β(t)，从 0.05 逐渐增加到 0.5，保持早期更新锚定在离线数据上。

- 在每一步：① 观测状态 st，编码为 zt；② 使用 DP 和高斯分布生成 Hmpc 步动作候选；③ 在潜在模型中滚动，计算累计奖励 + 终端 V；④ MPPI 迭代优化；⑤ 执行第一个动作；⑥ 将整条轨迹存入在线缓冲区；⑦ 从混合缓冲区采样，更新世界模型、评论家，以及 DP（通过 BC 蒸馏）。

## 6. 主要结论与发现

1. **MODIP 在多数任务上优于或持平于现有 DP 微调方法（DQL、DPPO、DSRL、PA-RL）和强模型基线 TD-MPC2**。尤其在 Kitchen 任务上表现突出（Complete 0.94，Partial 0.98）。
2. **MPC-to-policy 蒸馏有效**：用 MPC 改进轨迹后蒸馏到 DP，比直接 RL 微调更稳定，且能显著提升性能。
3. **终端状态值 V(s) 优于 Q(s,π(s))**：不仅推理时间降低 2.9 倍（85.75s → 29.54s per 10³ steps），性能也全面更好。
4. **策略无关评论家大幅降低训练时间**（约 1.6 倍加速）。
5. **DP 作为 MPC 先验优于 MLP 先验**：在 Hopper 和 Kitchen 任务上性能差距明显。
6. **离线预训练提供重要初始化**：尤其对早期规划质量影响大的任务（如 Hopper）有帮助。
