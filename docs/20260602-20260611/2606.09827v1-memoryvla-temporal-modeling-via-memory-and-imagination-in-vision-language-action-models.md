## 摘要
时间建模对于机器人操作至关重要，因为有效控制需要过去交互的记忆和未来状态的想象。然而，大多数VLA模型主要依赖当前观察，因此在长时间跨度、时间依赖的任务中表现不佳。认知科学表明，人类依赖工作记忆缓冲短期上下文，海马系统保存过去经历的情景记忆，以及内部模型想象可能的未来状态演化。受这些机制启发，我们提出MemoryVLA++，一个完整的时间建模框架，为机器人操作的VLA模型配备记忆和想象。预训练的VLM将当前观察编码为感知和认知令牌，形成工作记忆。这些令牌查询感知-认知记忆库以检索相关历史上下文。该库存储过去交互的低级细节和高级语义，并通过冗余感知合并进行更新。一个世界模型在去噪潜在空间中想象未来状态，然后在记忆引导下集成想象的潜在表示，形成完整的时间感知令牌。所得令牌条件化一个扩散动作专家，以预测时间一致的动作序列。我们在5个模拟基准和3类真实机器人任务（涉及3个机器人）上进行广泛实验，涵盖一般操作、长时间跨度时间任务、鲁棒性和泛化。我们的方法在Libero、SimplerEnv、Mikasa-Robo、Calvin、Libero-Plus以及多样化的真实机器人任务上取得了强劲性能，验证了具有记忆和想象的完整时间建模的有效性。例如，在真实机器人上，在一般、记忆依赖和想象依赖任务上分别获得了+9%、+26%、+28%的提升。项目页面：https://shihao1895.github.io/MemoryVLA-PP-Web

## Abstract
Temporal modeling is essential for robotic manipulation, as effective control requires both memory of past interactions and imagination of future states. However, most VLA models rely primarily on the current observation and therefore struggle with long-horizon, temporally dependent tasks. Cognitive science suggests that humans rely on working memory to buffer short-lived context, the hippocampal system to preserve episodic memory of past experience, and internal models to imagine possible future state evolution. Inspired by these mechanisms, we propose MemoryVLA++, a full temporal modeling framework that equips VLA models with memory and imagination for robotic manipulation. A pretrained VLM encodes the current observation into perceptual and cognitive tokens, forming working memory. These tokens query a Perceptual-Cognitive Memory Bank to retrieve relevant historical context. This bank stores low-level details and high-level semantics from past interactions, and is updated through redundancy-aware consolidation. A world model imagines future states in a denoising latent space, and the imagined latents are integrated under memory guidance to form full temporal-aware tokens. The resulting tokens condition a diffusion action expert to predict temporally consistent action sequences. We conduct extensive experiments on 5 simulation benchmarks and 3 categories of real-robot tasks across 3 robots, covering general manipulation, long-horizon temporal tasks, robustness, and generalization. Our method achieves strong performance across Libero, SimplerEnv, Mikasa-Robo, Calvin, Libero-Plus, and diverse real-robot tasks, validating the effectiveness of full temporal modeling with memory and imagination. For example, on real robots, it achieves +9%, +26%, +28% gains on general, memory-dependent, and imagination-dependent tasks. Project Page: https://shihao1895.github.io/MemoryVLA-PP-Web

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的视觉-语言-动作（VLA）模型（如 OpenVLA、π0）在机器人操作中仅依赖当前观测，缺乏对过去交互的记忆和未来状态演化的想象，因此在长时间跨度（long-horizon）和时间依赖的任务中表现不佳。
- **背景**：认知科学表明，人类依赖工作记忆（短期缓冲）、海马系统（情景记忆，存储过去经验）和内部模型（想象未来状态）来完成操作任务。
- **整体含义**：受此启发，作者提出 MemoryVLA++，将记忆和想象机制引入 VLA 模型，实现“过去-现在-未来”完整的时间建模，从而在通用操作、记忆依赖、想象依赖任务上显著提升性能。

## 2. 方法论：核心思想与关键技术细节

### 2.1 核心思想
- 采用 **工作记忆（Working Memory）** 编码当前观测（感知+认知 token）；
- 引入 **感知-认知记忆库（PCMB）** 存储历史交互的低级细节和高层语义，通过检索-融合-合并实现长期记忆；
- 利用 **世界模型**（基于 Stable Video Diffusion）在潜在空间进行部分去噪，生成未来想象 token；
- 在 **记忆引导** 下集成想象 token，形成完整时间感知表示；
- 最终用 **扩散动作专家**（DiT）预测时间一致的动作序列。

### 2.2 关键技术细节
- **视觉-语言-认知模块**：基于 7B Prismatic VLM（DINOv2 + SigLIP 编码器），产生感知 token（Nₚ×dₚ）和认知 token（1×dₑ）。
- **PCMB**：维护 L 个时序条目，每个条目包含感知（低层细节）和认知（高层语义）两部分。
  - **检索**：当前 token 作为 query，通过 cross-attention（加时间步位置编码）从 PCMB 中获取历史嵌入。
  - **门控融合**：由 sigmoid 门控向量自适应融合当前与检索结果：`x̃ = g⊙H + (1−g)⊙x`。
  - **合并更新**：当条目超过容量 L 时，合并最相似的相邻条目（余弦相似度），保持记忆紧凑。
- **世界模型想象**：
  - 采用预训练的 Stable Video Diffusion (SVD)，在机器人视频上微调（40k 步），冻结后用于推理。
  - 在潜在空间执行部分去噪（仅 1 步），通过 FPN 聚合多尺度 UNet 特征，经 Spatial Attention + Temporal Attention 压缩为想象 token z_img（K×Nₐ×dₚ）。
- **记忆引导想象集成**：记忆增强的感知 token 通过 cross-attention 与想象 token 交互，再通过门控融合：`p̄ = g⊙p̃ + (1−g)⊙h`。
- **扩散动作专家**：采用 DDIM（10 步采样），CFG 尺度 1.5。认知 token 与噪声动作序列进行 CogAttn，再与感知 token 进行 PerAttn，最后输出 7-DoF 动作序列。

  - 真实机器人：通用任务 +9%，记忆依赖 +26%，想象依赖 +28%。
- **记忆与想象缺一不可**：仅记忆（MemoryVLA）在想象依赖任务上不足（+16 vs +28），加上想象（MemoryVLA++）显著提升。
- **时间建模有效性**：在需要区分相似状态（如按钮按前/后）和需要预测运动（如传送带抓取）的任务上提升最大。

## 7. 优点

- **认知启发的完整时间建模**：系统性地结合工作记忆、情景记忆和内部模型，而非简单堆叠帧。
- **高效未来想象**：在潜在空间部分去噪，避免像素级预测，计算成本可控（推理延迟仅增加 0.05s）。
- **端到端可训练**：VLM、PCMB、世界模型、扩散动作专家联合优化（世界模型冻结），无复杂多阶段 pipeline。
- **极强的实验验证**：跨越 3 种机器人、5 个模拟基准、3 类真实任务，共近 200 个变体，消融全面。
- **开源与可复现**：提供了项目主页和视频结果。

## 8. 不足与局限

- **世界模型依赖**：需要额外训练世界模型（40k/20k 步），且在域外场景可能产生不准确想象，虽通过门控抑制但仍存在风险。
- **计算开销**：推理时多了世界模型前向（约 0.24s vs 0.19s），对于高频控制场景可能不足；GPU 内存增加至 21.7 GB（+5.9 GB）。
- **实验覆盖局限**：
  - 未在更多样化的真实环境（如动态障碍、非规则物体）或更长的任务链（>10 步）上测试。
  - 记忆长度消融中，长记忆在某些任务上反而下降（Libero-90 持平），提示需自适应调整容量。
- **动作输出维度相对固定**：仅验证了 7-DoF 单臂和双臂，未在人形机器人或更高自由度系统上测试。
- **泛化风险**：训练数据主要来自 BridgeData 和自收集，跨任务泛化能力仍需进一步检验（Libero-Plus 零样本仅 73%）。
