## 摘要
当前的视觉-语言-动作（VLA）模型通常针对语义基础进行优化，而可执行的操控任务需要几何感知的空间对齐和动态的可供性选择。我们提出GeoAlign，一种用于VLA策略学习的状态引导空间对齐架构。GeoAlign通过机器人域的RGB-D监督后训练一个RGB几何分支，为策略部署生成RGB衍生的几何增强后训练（GEP）特征。机器人的本体感知状态查询GEP特征网格，生成紧凑的、与阶段相关的几何令牌用于动作预测。GeoAlign在LIBERO上达到99.0%，在三个SimplerEnv-Fractal任务上平均85.3%，在八个几何关键的真实世界ALOHA任务上达到78.8%，消融实验证实了几何后训练和本体感知状态引导查询的价值。

## Abstract
Current Vision--Language--Action (VLA) models often optimize for semantic grounding, whereas executable manipulation requires geometry-aware spatial alignment and dynamic affordance selection. We introduce GeoAlign, a state-guided spatial alignment architecture for VLA policy learning. GeoAlign post-trains an RGB geometry branch with robot-domain RGB-D supervision, yielding RGB-derived Geometry-Enhanced Post-Trained (GEP) features for policy rollout. The robot's proprioceptive state queries the GEP feature grid, producing compact, phase-dependent geometry tokens for action prediction. GeoAlign achieves 99.0% on LIBERO, 85.3% across three SimplerEnv-Fractal tasks, and 78.8% on eight geometry-critical real-world ALOHA tasks, with ablations confirming the value of geometry post-training and proprioceptive-state-guided querying.

## 2. 方法论：核心思想、关键技术细节

### 2.1 整体架构（两阶段）

- **阶段一：离线几何后训练**：使用机器人域的RGB-D数据后训练Depth Anything V2-Small模型，优化尺度不变对数（SiLog）损失。训练后丢弃深度预测头，保留编码器侧特征作为**几何增强后训练（GEP）特征**。该特征在后续策略推理中从未使用原始深度预测。
- **阶段二：策略训练与部署**：冻结后训练的几何编码器，多视图RGB图像经VLM编码器生成语义令牌，同时经几何编码器生成图像空间GEP特征网格。**本体状态（本体感知状态，如关节位置）生成查询槽**，通过交叉注意力从GEP网格提取紧凑几何令牌，与语义令牌拼接后输入流匹配DiT动作解码器。

### 2.2 关键技术细节

- **GEP特征提取**：
  - 网格：每视图5,476个空间令牌（维度256），附加2D位置编码和视角嵌入。
  - 几何编码器输入：RGB图像，大小518×518，patch大小14，通过reassemble factor 2得到74×74特征图。
- **状态引导空间对齐**：
  - 本体状态经层归一化和编码器得到单个状态嵌入，再经MLP生成K=8个查询槽，加上可学习位置编码。
  - 8查询槽与GEP特征网格进行8头交叉注意力+FFN，输出8个紧凑几何令牌（投影到解码器维度2048）。
- **动作生成与训练**：
  - 语义令牌 Z_{vlm} 与几何令牌 Z_{geo} 拼接作为解码器上下文。
  - 解码器采用GR00T流匹配DiT动作头，使用beta时间采样器，训练损失为动作掩码归一化的均方误差（预测速度与目标速度）。
  - 推理时，从高斯噪声初始化，通过4步欧拉积分生成动作。

### 2.3 算法流程（文字描述）

```
1. 编码语义上下文：Z_vlm ← E_vlm(I_t, 语言指令)
2. 提取GEP特征：F_geo ← E_geo(I_t)，投影并添加位置/视角嵌入得到Φ_geo

```
