## 摘要
视觉世界模型在学习复杂系统动力学方面展现出巨大潜力。最近的研究进展将这些模型用作模型预测控制（MPC）框架中的转移函数，以解决各种控制任务。然而，当应用于机器人领域时，它们仅限于单阶段任务（如到达或抓取），难以应对需要复杂序列规划的多阶段任务。在这项工作中，我们介绍WorldDP，一个专为多阶段机器人操作设计的世界模型框架。我们的层次化方法利用高层世界模型作为转移函数，在运行时优化可行的子目标，随后由低层扩散策略执行这些子目标。为进一步辅助学习动力学和规划，我们引入了以物体为中心的表示，将环境实体解耦，使得能够针对每个实体进行序列规划。在多个机器人基准测试中的评估表明，WorldDP始终优于现有基线，验证了将世界模型的物理基础规划与扩散策略的高效执行相结合，能产生更优越的多阶段性能。

## Abstract
Visual world models have shown great potential in learning complex system dynamics. Recent advancements leverage these models as transition functions within Model Predictive Control (MPC) frameworks to solve various control tasks. When applied to robotics, however, they are limited to single-stage tasks such as reaching or grasping, and struggle with multi-stage ones that demand complex sequential planning. In this work, we introduce WorldDP, a world model framework designed for multi-stage robotic manipulation. Our hierarchical approach utilizes a high-level world model as a transition function to optimize for feasible subgoals during runtime, which are subsequently reached by a low-level Diffusion Policy. To further aid in learning dynamics and planning, we incorporate object-centric representations that decouple environmental entities and enable us to plan sequentially with respect to each. Evaluated across several robotics benchmarks, WorldDP consistently outperforms existing baselines, validating that coupling the world model's physically grounded planning with diffusion policy's efficient execution yields superior multi-stage performance.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 采用**层次化结构**：高层世界模型作为转移函数，在运行时通过**粒子滤波器**优化生成可行的**子目标**；低层由**目标条件扩散策略**（Goal-Conditioned Diffusion Policy）逐步跟踪并执行这些子目标。
- 引入**以物体为中心（Object-Centric）的表示**，将环境中的机器人、物体和背景解耦为独立槽（slots），使世界模型能够针对每个物体分别进行规划和动力学建模。
- 增加**接触预测器**，引导子目标生成时关注关键交互帧（如抓取瞬间），并整合到代价函数中。

### 关键技术细节

1. **物体中心编码器（OCE）**  
   - 输入图像经过冻结的 DINOv2 编码器得到 patch‑级特征。  
   - 采用 **Slot Attention** 框架初始化 N 个槽，通过交叉注意力和 GRU 校正器迭代得到物体中心状态表示 \(s_k \in \mathbb{R}^{N \times d}\)。  
   - 训练时使用 Slot Decoder 重构 patch 特征和掩码，损失函数为重构损失 \(L_{l2\_oce}\) 与 Tversky 损失 \(L_{tv}\)（利用 SAM2 生成的ground‑truth 掩码引导），最终损失 \(L_{oce} = L_{l2\_oce} + \lambda_{oce} L_{tv}\)。

2. **条件扩散变换器动力学模型**  
   - 基于 CDiT 架构，输入物体中心状态 \(s_k\)，动作序列通过 Transformer 编码为 32 维潜在动作嵌入，以自适应层归一化条件注入。  
   - 训练时随机采样 5 帧（时间间隔可变），以教师强制方式预测下一状态，使用 MSE 损失。

3. **任务执行与子目标优化**  
   - **粒子滤波器**替代传统的 CEM：初始化 Q 个粒子（从空间轨迹种子和随机 yaw/gripper 生成），每次迭代围绕粒子均值采样动作序列，用代价函数评估后保留 Top‑M 作为新均值，迭代 L 轮。  
   - **代价函数**：  
     \[
     C(\hat{s}_T, s_g) = C_{dist} + \lambda_{plan} \cdot C_{ce}(c, c_{ref})
     \]  
     其中 \(C_{dist}\) 为物体嵌入的 MSE，\(C_{ce}\) 为接触预测器输出的交叉熵损失，\(c_{ref}\) 为目标物体的 one‑hot 向量。  
   - **层次执行**：对多物体任务，依次规划并执行每个物体的子目标序列，形成 MPC 循环。

4. **低层扩散策略**  
   - 基于 Transformer 的扩散策略，输入当前及目标物体中心状态（展平向量）和末端位姿/速度，输出 40 步动作序列。  
   - 使用随机目标采样策略增强时间泛化，并通过深度相机修正抓取姿态。

## 3. 实验设计

### 数据集与任务场景（OGBench 扩展版）
- **Cube‑Single**：单红方块，需从初始位置移动到目标位置。  
- **Cube‑Triple**：三个不同色方块，需按目标配置依次重排。  
- **Scene‑Single‑Direct**：包含抽屉、滑动窗、方块和两个按钮，起始与目标差异仅涉及一个实体（按钮、抽屉或窗）。  
- **Scene‑Single‑Composite**：相同环境，但必须先按相应按钮解锁才能操作抽屉/窗，考验因果依赖。

### 训练数据
每个环境收集 **200 万帧** “play” 数据集（无奖励、无目标标签，含次优轨迹）。图像大小 224×224。

### 对比基线
- **DINO‑WM** (Zhou et al., 2024)：基于冻结 DINOv2 patch 特征的 JEPA 世界模型。  
- **LeWM** (Maes et al., 2026b)：联合优化编码器和潜在动力学模型。  
- **HECRL***：改编自 Haramati et al. (2026)，将低层 RL 替换为相同扩散策略，单视图输入。  
- **DP40 / DP100**：标准目标条件扩散策略，分别训练 40 步和 100 步轨迹。

### 评估设置
- 每个任务使用 50 条独立测试轨迹（所有方法相同）。  
- 成功率定义：方块距离 <12 cm，按钮状态匹配，抽屉/窗位置误差 <4 cm / 3 cm。  
- 报告 Cube‑Triple 的 1‑cube、2‑cube、3‑cube 成功率；Scene‑Single‑Composite 的按钮按压和完整任务成功率。

## 4. 资源与算力

论文明确提到：所有模型在**单张 NVIDIA H100 GPU 集群节点**上训练。OCE 训练 10 轮，batch size 64；动力学模型训练 10 轮，batch size 128；扩散策略训练 3 轮，batch size 48；接触预测器训练 10 轮，batch size 256。优化器均为 AdamW，学习率采用 one‑cycle 余弦退火调度。没有报告具体训练时长，但基于 200 万帧和 batch size 128，每轮约 15,500 步，5 轮约 77,500 步，在单卡 H100 上可合理完成。

## 6. 论文的主要结论与发现

- WorldDP 在所有多阶段任务上**一致优于**所有基线。  
  - Cube‑Triple 中 3‑cube 成功率 30%，远高于次佳方法（HECRL* 12%）。  
  - Scene‑Single‑Composite 完整任务成功率 20%，而其他基线均低于 18%。  
  - Scene‑Single‑Direct 平均成功率 74%，显著领先第二（HECRL* 63%）。  
- 层次化融合世界模型规划与扩散策略执行是成功关键：去除 DP 后性能大幅下降；使用 40 步扩散策略优于 100 步，因为世界模型生成密集子目标更适合短程执行。  
- 物体中心编码显著提升复杂场景（如 Cube‑Triple 和 Scene‑Single）的表现。  
- 粒子滤波器和接触预测器均对结果有正向贡献。  

## 7. 优点

1. **新颖的层次化框架**：将世界模型的长视野规划能力与扩散策略的短时鲁棒执行能力有机结合，克服了各自短板。  
2. **物体中心表示**：解耦环境实体，实现对每个物体的单独规划，更加符合物理世界直觉，且比 patch‑级表示保留更多细节；配合 SAM2 引导训练，提升小物体感知。  
3. **粒子滤波器应用**：首次在世界模型规划中引入粒子滤波器，相比 CEM 能更好捕捉机器人动作的多模态性。  
4. **接触预测器**：以辅助损失形式引导子目标生成，鼓励模型产出关键交互帧，提升任务成功率。  
5. **无需奖励/目标标签**：仅需次优 play 数据集，泛化能力强。  
6. **实验设计全面**：消融实验详细，对比方法多样，结果统计可靠。
