## 摘要
视觉-语言-动作（VLA）模型在机器人操作中展现出强大潜力，但其测试时的可靠性仍受限于单次动作预测，即使微小的动作误差也可能导致抓取失败、碰撞或任务进度错误。一个自然的替代方案是为VLA系统配备测试时验证，使其在执行前能够提出并评估多个候选动作。然而，可靠的动作验证具有挑战性，因为它不仅需要区分候选动作之间细微的几何差异，还需要评估该动作是否朝着任务目标取得有意义的进展。我们提出了VeriSpace，一种用于VLA系统测试时动作选择的3D感知动作验证器。VeriSpace通过两个关键组件评估候选动作：双路径3D注入场景编码，该编码构建了同时保留视觉语义和显式3D几何的场景表示；以及空间化动作推理，该推理通过推理任务相关的空间关系、几何有效性和预期的目标进展来评估每个动作。这些组件共同实现了对微妙但结果关键的候选动作的更可靠区分，同时与现有的VLA策略完全兼容。在公共基准和真实世界机器人操作任务上的实验表明，VeriSpace在分布内和分布外设置中均能持续提高底层VLA策略和先前基于验证的方法的决策可靠性，带来显著的性能提升。

## Abstract
Vision-language-action (VLA) models have shown strong promise for robotic manipulation, but their reliability at test time remains limited by one-shot action prediction, where even small action errors can cause grasp failure, collision, or incorrect task progression. A natural alternative is to equip VLA systems with test-time verification, allowing multiple candidate actions to be proposed and evaluated before execution. However, reliable action verification is challenging because it requires not only distinguishing subtle geometric differences between candidate actions, but also assessing whether an action makes meaningful progress toward the task goal. We present VeriSpace, a 3D-aware action verifier for test-time action selection in VLA systems. VeriSpace evaluates candidate actions through two key components: Dual-Path 3D-Injected Scene Encoding, which constructs a scene representation that jointly preserves visual semantics and explicit 3D geometry, and Spatially-Grounded Action Reasoning, which evaluates each action by reasoning over task-relevant spatial relations, geometric validity, and expected goal progress. Together, these components enable more reliable discrimination between subtle yet outcome-critical action candidates while remaining fully compatible with existing VLA policies. Experiments on public benchmarks and real-world robotic manipulation tasks show that VeriSpace consistently improves decision reliability over both underlying VLA policies and prior verification-based methods, yielding substantial gains in both in-distribution and out-of-distribution settings.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：现有的视觉-语言-动作（VLA）模型在测试时通常采用**单次动作预测**（one-shot prediction），即直接执行模型输出的唯一动作。由于机器人操作对微小误差极为敏感，这种“一次性”决策方式容易导致抓取失败、碰撞或任务进度错误，尤其在非结构化的精细场景中。
- **整体含义**：为了让VLA系统更加可靠，应引入**测试时验证**（test-time verification）机制：先由策略模型采样多个候选动作，再通过一个验证器（verifier）评估每个动作的空间合理性和对任务目标的推进程度，最后选择最佳动作执行。这种“生成-验证”范式是提升机器人鲁棒性的重要方向。
- **背景**：已有工作（如V‑GPS、Robomonkey、MG‑Select）尝试用预训练VLM直接对动作打分，但主要依赖2D图像，难以区分细微的空间差异（如夹爪是否对准物体中心）。因此，论文提出结合**3D几何感知**和**结构化空间推理**的验证器。

## 2. 论文提出的方法论
### 2.1 核心思想
- 提出**VeriSpace**，一个**3D感知的动作验证器**，作为插件式模块叠加在已有VLA策略之上。它不修改策略的训练或架构，只在测试时对多个候选动作进行评分和排序。
- 核心思想：通过**双路径3D注入场景编码**（Dual‑Path 3D‑Injected Scene Encoding）同时保留视觉语义和显式3D几何；通过**空间化动作推理**（Spatially‑Grounded Action Reasoning）以链式思维评估动作的空间关系、几何有效性和任务进展。

### 2.2 关键技术细节
1. **测试时验证设置**  
   - 给定当前观察（RGB‑D + 手腕图像）、机器人状态和语言指令，从VLA策略中以温度τ采样k个候选动作（7维：6‑DoF末端运动+夹爪动作）。  
   - 验证器F_φ为每个动作输出标量得分，选择得分最高的动作执行。

2. **双路径3D注入场景编码**  
   - **3D几何构建**：利用已知相机内外参将RGB‑D反投影为3D坐标图P∈ℝ^(H×W×3)，经正弦编码+MLP得到几何嵌入P′。  
   - **双路径令牌化**：  
     - 3D几何令牌 X^geo：将P′投影到VLM令牌空间，提供几何中心表示。  
     - 空间增强视觉令牌 X^vis：将CLIP视觉令牌与P′逐元素相加，保留语义的同时注入3D位置。  
   - **几何引导局部聚合（GLA）**：对X^vis使用最远点采样（FPS）选种子令牌，在3D球半径内按相对偏移加权聚合邻近令牌，增强局部几何结构捕获能力。  
   - 最终令牌序列 = [X^vis, X^geo, 指令令牌, 动作令牌] 送入VLM。

3. **空间化动作推理**  
   - 采用链式推理（CoT）进行三步评估：  
     - **步骤1：动作空间推理**——定位操作对象、目标对象、机器人末端，分析相对位置、朝向、接触、间隙。  
     - **步骤2：目标进展评估**——分两个子视角：  
       - 动作前评估：当前场景相对于目标状态的关系。  
       - 动作后评估：推断动作执行后的场景变化，比较是否更接近目标。  
     - **步骤3：验证得分估计**——整合上述推理结果输出标量得分。  
   - 推理过程通过监督学习训练，CoT数据由深度估计（Metric Depth Anything V2）、物体检测（Grounding DINO）、空间关系推理（SpatialRGPT）自动生成。

4. **学习目标**  
   - 动作评价损失 L_a：采用Bradley‑Terry偏好对，鼓励优质动作得分高于劣质动作。  
   - 推理损失 L_CoT：交叉熵损失，监督CoT序列生成。  
   - 总损失 L = λ_a L_a + λ_CoT L_CoT。
