## 摘要
世界动作模型（WAMs）已成为具身智能的一种新范式，通过学习与动作相关的视觉动态，显著增强了泛化能力和鲁棒性。然而，现有WAMs在长时程机器人操作中仍难以处理任务相关的记忆问题。为此，我们提出HiMem-WAM，一种分层记忆门控WAM，它融合了以运动为中心的潜在动作、高层技能潜在变量以及边界触发的记忆更新。具体而言，我们开发了一个分层潜在动作框架，联合学习底层运动和高层技能潜在变量，提供结构化的时间抽象。同时，一个边界感知记忆门在预测的技能转换处写入紧凑的任务状态，从而实现因果推理，无需在测试时生成未来视频或光流估计。在LIBERO、LIBERO-PLUS、RMBench以及真实世界任务上的评估表明，HiMem-WAM的分层潜在变量提高了部署扰动下的鲁棒性，且记忆模块显著有利于记忆依赖的长时程操作。

## Abstract
World Action Models (WAMs) have emerged as a new powerful paradigm for embodied intelligence, learning action-relevant visual dynamics that significantly enhance generalization and robustness. However, existing WAMs still struggle with task-relevant memory in long-horizon robotic manipulation. To address this, we present HiMem-WAM, a Hierarchical Memory-Gated WAM that integrates motion-centric latent actions, high-level skill latents, and boundary-triggered memory updates. Specifically, we develop a hierarchical latent action framework that jointly learns low-level motion and high-level skill latents, providing structured temporal abstraction. Meanwhile, a boundary-aware memory gate writes compact task states at predicted skill transitions, enabling causal inference without test-time generation of future video or optical flow estimation. Evaluated on LIBERO, LIBERO-PLUS, RMBench and real-world tasks, HiMem-WAM shows that hierarchical latents improve robustness under deployment perturbations, and the memory module substantially benefits memory-dependent long-horizon manipulation.

## 2. 论文提出的方法论

### 2.1 核心思想
- **层次化潜在动作框架**：将动作表示分解为两个层次：
  - **低层潜在动作（Low-level latent action, \(z^l\)）**：捕捉局部运动（如夹爪移动、旋转），通过光流和动作标签学习紧凑的运动空间。
  - **高层技能潜在变量（High-level skill latent, \(z^h\)）**：通过动态分割低层潜在序列发现技能边界，每个技能对应一个高层潜在，提供结构化时间抽象。
- **门控记忆模块**：在预测的技能转换边界处，通过“读门”和“写门”控制记忆的读取与写入。写门仅在检测到技能边界（由边界预测器输出）时触发，写入紧凑的任务状态令牌；读门从外部记忆库检索历史上下文，并与当前状态融合，用于规划器。

### 2.2 关键技术细节
- **低层潜在分词器**：使用DPFlow提取多视角光流，结合当前帧观测、下一帧、本体感知、语言指令，通过变分编码器编码为高斯分布，并用KL散度和动作对齐损失（动作标注可用时）训练。
- **高层技能发现**：通过层次化分块（Chunking）算法，对低层潜在序列逐层进行边界预测（基于query-key相似度不一致）、变长段池化（注意力池化），最终得到技能序列\(z^h\)及逐时间步的边界标签\(\bar{b}_t\)。
- **规划器与执行器**：
  - 规划器：基于Qwen3-VL-4B-Instruct，输入观测、语言、本体、记忆上下文，预测当前技能\(z^h_t\)和边界分数\(\hat{b}_t\)。
  - 执行器：从记忆增强状态\(\tilde{x}_t\)和技能\(z^h_t\)出发，预测低层潜在动作块\(\hat{Z}^l_{t:t+K-1}\)，再由动作解码器映射为实际控制信号。
- **记忆读取与写入**：

  - 写入：\(\alpha^w_t = \sigma(G_w(\tilde{x}_t, z^h_t, \hat{b}_t))\)，若大于阈值\(\eta\)，则将候选记忆令牌\(\gamma_t\)（由状态、技能、低层潜在池化构成）写入记忆库。
- **三阶段训练流水线**：
  - Stage I：训练低层分词器（冻结后离线提取全部轨迹的\(z^l\)序列）。
  - Stage II：固定分词器，学习技能边界，预训练规划器和执行器（无记忆）。
  - Stage III：激活记忆模块，使用动作标注数据微调整个策略（包括门控网络）。
