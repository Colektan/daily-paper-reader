## 摘要
我们提出Q引导的价值梯度匹配（Q-VGM），一种离策略强化学习方法，旨在解决微调流匹配视觉-语言-动作（VLA）策略中的一个长期挑战：如何高效地利用学习的Q函数改进一个表达能力强的流匹配动作专家。有效的改进必须利用评论家的一阶（梯度）信息，但这对于流策略而言较为困难，因为直接通过其多步去噪过程反向传播价值在VLA规模下数值不稳定，而策略梯度方法所需的可操作动作似然在迭代去噪下不可用。现有的基于价值的方法要么通过整个去噪链进行反向传播，要么仅在测试时使用评论家而不更新策略，要么将评论家改进的动作作为终端标签进行蒸馏而不对速度场进行监督。Q-VGM通过利用VGG-Flow（一种生成式建模中流对齐的价值梯度视角）规避了这些问题，该视角将价值梯度转化为一个去噪时间价值梯度场，而非不稳定的端到端目标。该方法不需要动作似然，也不需要反向传播通过去噪链，并且在固定重放缓冲区上运行。评论家是一个基于紧凑RLT特征的动作敏感Cal-QL集成，具有逐层动作注入。Q-VGM实现了一种实用的少样本初始化然后从经验中学习的范式：从少样本监督微调的pi0.5 VLA开始，该方法利用自生成的回滚数据显著提升任务性能，而无需额外的专家监督。在LIBERO上，Q-VGM将平均成功率从75.0%提升至92.5%；在RoboTwin 2.0上，从76.4%提升至87.2%；在两个真实机器人桌面任务上，从40.0%提升至67.5%，在所有三个设置中均优于所有相同骨干网络和相同评论家的基线方法。

## Abstract
We propose Q-Guided Value-Gradient Matching (Q-VGM), an off-policy reinforcement learning (RL) method that tackles a long-standing challenge in fine-tuning flow-matching vision-language-action (VLA) policies: efficiently improving an expressive flow-matching action expert with respect to a learned Q-function. Effective improvement must exploit the first-order (gradient) information of the critic, but this is difficult for flow policies, because directly back-propagating the value through their multi-step denoising process is numerically unstable at VLA scale, while the tractable action likelihoods required by policy-gradient methods are unavailable under iterative denoising. Existing value-based methods either backpropagate through the full denoising chain, use the critic only at test time without updating the policy, or distill critic-improved actions as terminal labels without supervising the velocity field. Q-VGM sidesteps these issues by leveraging VGG-Flow, a value-gradient view of flow alignment in generative modeling that transforms value gradient into a denoising-time value-gradient field rather than an unstable end-to-end objective. This requires no action likelihoods and no backpropagation through the denoising chain, and operates on a fixed replay buffer. The critic is an action-sensitive Cal-QL ensemble over compact RLT features with per-layer action injection. Q-VGM enables a practical few-shot initialization then learn-from-experience paradigm: starting from a few-shot-SFT pi0.5 VLA, the method leverages self-generated rollout data to substantially improve task performance without additional expert supervision. On LIBERO, Q-VGM raises the average success rate from 75.0% to 92.5%; on RoboTwin 2.0, from 76.4% to 87.2%; and on two real-robot tabletop tasks, from 40.0% to 67.5%, outperforming all same-backbone, same-critic baselines across all three settings.

## 1. 核心问题与整体含义（研究动机和背景）
针对流匹配（Flow-Matching）视觉-语言-动作（VLA）策略的强化学习微调，现有方法面临三个核心困难：
- **梯度不稳定**：直接反向传播Q函数通过多步去噪链（如Diffusion-QL）在VLA规模下数值不稳定；
- **似然不可得**：流策略通过迭代去噪生成动作，不暴露可操作的似然，因此策略梯度方法不可用；
- **监督不完整**：现有基于价值的方法要么仅在测试时使用评论家（不更新策略），要么将批评家改进的动作作为终端标签蒸馏（忽略速度场结构）。

**整体意义**：Q-VGM 提出一种离策略（off-policy）价值引导微调方法，通过将价值梯度转化为去噪时间步上的速度校正，无需似然和反向传播去噪链，使流匹配VLA策略能从自身回滚数据中持续改进，提升任务成功率。

## 2. 方法论：核心思想、关键技术细节
### 2.1 核心思想
基于VGG-Flow的最优控制视角——最优速度校正与去噪时间价值函数的梯度成正比。Q-VGM 将评论家梯度（定义在干净动作上）映射为去噪过程中的速度校正，通过残差速度匹配训练策略。

### 2.2 关键技术步骤
1. **动作敏感评论家（Action-Sensitive Chunk Critic）**：
   - 状态表示：使用RLT编码器从冻结VLM骨干提取紧凑特征向量 \(z_{rl}\)，与投影本体感觉拼接；
   - 逐层动作注入：在每个隐藏层重新拼接动作块，保持对动作变化的敏感性；
   - Cal-QL集成：多个Q头（ensemble），使用Cal-QL目标训练，确保动作梯度平滑。

2. **Q引导的价值梯度匹配**：
   - **前向估计（Look-forward clean-action estimate）**：对每个去噪状态 \(x_k\)，通过一步欧拉投影（使用冻结基础速度）得到干净动作估计 \(\hat{A}_{k,\text{base}}\)；
   - **迭代Q梯度上升**：从 \(\hat{A}_{k,\text{base}}\) 出发，进行 \(J\) 步带裁剪的梯度上升（公式6），生成候选动作；
   - **保留最佳（Keep-best selection）**：从所有候选（包括原始估计）中选择Q值最高的动作，作为自适应校正幅度；
   - **速度校正计算**：将动作改进转化为速度校正 \(h_{k}^{\text{eff}} = (\hat{A}_{k,\text{base}} - \hat{A}_{k,j^*}) / t_k\)；
   - **去噪时间门控**：使用 \(s(t_k) = (1 - t_k)^p\) 调节不同时间步的校正权重。

3. **残差速度匹配损失**：
   \[
   L_{\text{align}} = \sum_k s(t_k) \left\| \left(v_\theta(x_k,t_k,s) - v_{\text{base}}(x_k,t_k,s)\right) - \text{sg}[h_k^{\text{eff}}] \right\|_2^2
   \]
   梯度仅流入当前策略预测 \(v_\theta\)，其他项停止梯度。

4. **推理**：策略使用 \(v_\theta = v_{\text{base}} + h_\theta\) 采样，无需测试时搜索或反向传播。

### 2.3 算法流程简述
- 初始化：从少样本SFT π0.5开始
- 收集固定回滚数据
- 训练Cal-QL集成评论家（冻结VLM和RLT）
- 对于每个训练迭代：
  - 采样状态和初始噪声
  - 沿去噪链前向传播策略（stop-gradient）
  - 对每个时间步：前向估计 → 迭代梯度上升 → 保留最佳 → 计算速度校正
  - 通过残差速度匹配更新策略参数
