## 摘要
在动态环境中进行终身具身导航要求机器人从碎片化观测中形成持久场景理解，这对于依赖显式地图或场景图且难以泛化到结构化环境之外的现有方法而言仍然困难。我们提出AllDayNav，一种终身自学习导航框架，通过强化学习将场景动态隐式编码到大型模型的十亿级参数中，并由自我进化的多模态记忆驱动，该记忆维护和更新视觉关键帧、语义描述和时间上下文，同时自主生成开放词汇指令、图像目标和结构化奖励。在跨房间、跨回合和跨任务场景的合成及真实环境中的实验表明，AllDayNav的成功率接近100%，并在路径效率和鲁棒性上持续超越强大的基于地图、VLM和RL的基线方法，证明了隐式、记忆驱动的强化学习作为显式映射的可扩展替代方案，用于可靠的终身导航。

## Abstract
Lifelong embodied navigation in dynamic environments requires robots to form persistent scene understanding from fragmentary observations, which remains difficult for existing methods that rely on explicit maps or scene graphs and struggle to generalize beyond structured settings. We propose AllDayNav, a lifelong self-learning navigation framework that implicitly encodes scene dynamics into the billion-scale parameters of a large model via reinforcement learning, powered by a self-evolving multimodal memory that maintains and updates visual keyframes, semantic descriptions, and temporal context while autonomously generating open-vocabulary instructions, image goals, and structured rewards. Experiments in both synthetic and real-world environments across cross-room, cross-episode, and cross-task scenarios show that AllDayNav achieves success rates approaching $100\%$ and consistently surpasses strong map-based, VLM, and RL baselines in path efficiency and robustness, demonstrating implicit, memory-driven reinforcement learning as a scalable alternative to explicit mapping for reliable lifelong navigation.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想：记忆-策略共同进化（Memory-Policy Co-evolution）
- 机器人通过探索构建并持续更新多模态记忆数据库，该数据库同时用于生成自监督训练任务和检索视觉目标。机器人基于这些任务通过强化学习优化策略，而改进后的策略又能更高效地探索环境，从而形成良性循环。

### 关键技术细节

#### 2.1 预训练 VLA 导航骨干
- **架构**：基于 Qwen2-7B 的 Transformer，输入由三部分拼接：
  - 当前观测帧（使用 SigLIP + DINOv2 双编码器提取语义和结构特征，经 MLP 投影）
  - 目标图像（同样编码）
  - 历史观测帧（使用全局平均池化压缩为每帧 4 个 token，并施加指数衰减注意力偏置，窗口大小 64 帧）
- **动作表示**：预测 8 个未来局部路点（连续坐标 + 航向角），便于底层控制器执行。
- **训练**：在 Habitat 仿真器中收集 HM3D + MP3D 共 5 万条轨迹（80% 最短路径专家轨迹，20% 探索轨迹），使用 MSE 损失进行行为克隆，并应用强烈的图像增强（模糊、抖动、随机裁切等）以弥合 sim-to-real 差距。

#### 2.2 自演进多模态记忆数据库
- **条目结构**：RGB 关键帧 + VLM 生成的语义描述 + 时间戳 + LLaVE 视觉嵌入。
- **构建**：每 5 步检测一次；若新帧嵌入与上一个关键帧变化大于阈值（δ=0.15）则作为候选。
- **更新与去重**：候选帧与现有记忆进行余弦相似度比较：
  - 若最大相似度 ≤ 0.7，作为新条目插入（并调用 Qwen3-VL-8B 生成描述）。
  - 若 >0.7，则比较新图像与旧图像的文本-图像对齐分数，保留更优者。
- **语义检索**：给定自然语言指令，使用 VLM 在所有描述（含时间戳）中选出最匹配的场景，返回图像及置信度。

#### 2.3 自指令生成与混合目标检索
- **自指令生成**：每隔 10 回合或记忆增长 20% 时，将当前记忆中的描述汇总，输入 Qwen3-VL-8B 生成 50 条多样化指令（物体、功能、空间、组合、时间等）。
- **混合检索**：对每条指令，先尝试从记忆中检索目标图像；若置信度低于 0.6 或早期随机回退，则从互联网图片数据库（G_internet）中检索对应图像作为目标。

#### 2.4 保守 Q 学习（CQL）策略优化
- **奖励设计**：基于 LLaVE 嵌入的余弦相似度：终端奖励（相似度≥0.75 时 +10）、进度奖励（首次达到 0.5/0.6/0.7 时 +2）、时间惩罚（每步 -0.01）、碰撞惩罚（-0.1）。
- **CQL 损失**：在标准贝尔曼误差基础上增加保守正则化项，惩罚OOD动作的高 Q 值（α=5.0）。
- **策略损失**：最大化 Q 值 + 行为克隆正则项（λ=0.1），从成功轨迹中提取样本。
- **更新频率**：每 100 步更新一次。

### 算法流程（文字描述）
1. 初始化记忆数据库 M 为空，回放缓冲池 B 为空，评价网络 Q。
2. 并行运行两个进程：
   - **主动探索循环**：每回合从自指令集中采样指令 → 混合检索得到目标图像 → 执行策略 π_θ，收集轨迹并存入 B → 实时更新 M（新关键帧、去重、更新描述）。
   - **策略优化循环**：从 B 中采样批量 → 更新评价网络（CQL）→ 更新策略网络（结合BC损失）。
3. 随经验积累，性能持续提升。
