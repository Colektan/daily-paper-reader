## 摘要
世界-动作模型已成为机器人操作的一种有前途范式，它联合建模视觉场景动态和动作，从而将物理先验注入策略学习。然而，现有的世界-动作模型以相同的时间分辨率耦合世界预测和动作执行，迫使世界分支建模近期的帧变化，这些变化是冗余且信息量较弱的。我们假设，严格将世界预测和动作执行绑定到相同的时间节律可能会低估视频分支在具身控制中的潜力。因此，我们提出了AHA-WAM，一种基于双扩散Transformer（DiT）架构的异步自适应视界世界-动作模型，该模型围绕这种时间不对称性重新组织世界-动作建模。AHA-WAM将视频DiT实例化为一个低频世界规划器，它维护过去观测的滚动键值记忆，并暴露可复用的逐层潜在上下文编码，以捕捉长视场景演变，同时高频动作DiT通过逐层联合注意力查询该上下文，以闭环方式执行短动作块。为了支持异步执行，我们引入了自适应视界偏移训练和观测引导的视频上下文路由（OVCR），这两者共同使得动作专家能够利用长视界世界上下文，同时保持对实时执行状态的响应，而无需重新运行视频DiT。在RoboTwin和现实世界操作任务上的实验表明，AHA-WAM在没有任何机器人数据预训练的情况下实现了最先进的性能，在RoboTwin上平均成功率为92.80%，在4个现实世界任务上成功率为78.3%，同时达到了24.17 Hz的闭环控制速度，比Fast-WAM加速了4.59倍。

## Abstract
World-action models have emerged as a promising paradigm for robot manipulation, jointly modeling visual scene dynamics and actions to inject physical priors into policy learning. However, existing world-action models couple world prediction and action execution at the same temporal resolution, forcing the world branch to model near-term frame variations that are redundant and weakly informative. We posit that strictly binding world prediction and action execution to the same temporal rhythm may underutilize the potential of the video branch for embodied control. Therefore, we propose AHA-WAM, an Asynchronous Horizon-Adaptive World-Action Model built on a dual Diffusion Transformer (DiT) architecture that reorganizes world-action modeling around this temporal asymmetry. AHA-WAM instantiates the video DiT as a low-frequency world planner that maintains rolling key-value memory over past observations and exposes reusable layerwise latent context encoding long-horizon scene evolution, while a high-frequency action DiT executes short action chunks in closed loop by querying this context through layerwise joint attention. To support asynchronous execution, we introduce horizon-adaptive offset training and Observation-Guided Video-Context Routing (OVCR), which together let the action expert exploit long-horizon world context while remaining responsive to real-time execution state without rerunning the video DiT. Experiments on RoboTwin and real-world manipulation tasks show that AHA-WAM achieves state-of-the-art performance without any robot-data pretraining, attaining 92.80% average success on RoboTwin and 78.3% success across 4 real-world tasks, while reaching 24.17 Hz closed-loop control with a 4.59x speedup over Fast-WAM.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：现有的世界-动作模型（WAM）将世界预测（视频建模）与动作执行绑定在相同的时间分辨率下，导致视频分支被迫建模相邻帧之间高度相关、信息量冗余的短期变化，未能充分利用视频分支的长时序规划潜力，同时限制了高频闭环控制。
- **研究背景**：机器人操作需要同时理解当前场景及其在动作下的演化。视觉-语言-动作（VLA）模型通过大规模模仿学习取得进展，但动作标签对物理动态的监督稀疏。WAM通过联合建模动作与密集视频动态来注入物理先验，但现有方法（如Fast-WAM、Motus）都采用同步建模，使世界分支专注于短视界预测，与动作执行共享相同节律，造成结构性的时间抽象不匹配。
- **整体含义**：本文提出**时间不对称性**应被利用：视频世界模型应形成时间上扩展的潜在规划（低频、长视界），而动作模型应紧密耦合实时控制（高频、短视界）。AHA-WAM旨在解耦二者，在保持高性能的同时显著提升推理效率。

## 2. 论文提出的方法论

### 核心思想
- **异步双DiT架构**：将世界建模与动作执行解耦为两个不同时间尺度的扩散Transformer（DiT）：
  - **低频世界规划器**（Video DiT）：基于Wan2.2-5B视频模型初始化，以低频率处理长视界（h_v=64帧）的未来视频潜在编码，维护滚动KV记忆（Rolling K/V Memory）存储历史观测，输出可复用的逐层潜在上下文。
  - **高频动作专家**（Action DiT）：轻量级DiT（约1.02B参数），以高频率（h_a=16动作）闭环执行短动作块，通过逐层联合注意力（Joint Attention）从世界规划器查询上下文，同时接收本体感受反馈。
- **观察引导的视频上下文路由（OVCR）**：解决异步执行中规划器上下文可能过时的问题。OVCR利用最新视觉观测，通过可学习的基础查询对视频特征进行注意力池化，生成紧凑的观测引导查询，然后读取规划器缓存的KV对并预测残差更新（残差门控机制），从而为每个动作块产生观测修正的规划器上下文，无需重新运行视频DiT。
- **自适应视界偏移训练（Horizon-Adaptive Offset Training）**：在训练中随机偏移动作块在规划器视界内的起始位置（δ~U[0, h_a)），使动作专家适应异步部署中规划器-执行器之间的任意相位偏移，增强鲁棒性。
- **滚动KV记忆**：规划器内部维护一个固定大小的FIFO记忆，存储历史规划刷新时的视频状态，扩展规划器的时序感受野。

### 关键技术细节
- **联合流匹配目标**：同时优化动作流匹配损失L_a和视频流匹配损失L_v，最终损失L = L_a + λL_v。
- **注意力机制**：动作DiT通过层间联合注意力融合自身查询与经过OVCR更新的规划器KV对，实现视觉-动作交互。
- **推理优化**：使用TensorRT、CUDA Graph、热路径冗余消除等技术加速动作推理，并采用ODE蒸馏将动作去噪步数从10步减少到2步（AHA-WAM-Flash）。
