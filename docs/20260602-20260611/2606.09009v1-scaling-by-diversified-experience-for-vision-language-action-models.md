## 摘要
视觉-语言-动作模型由于高级推理与低级控制的纠缠以及策略优化的不稳定性，在实际部署中面临重大挑战。本文介绍SyVLA，一个通过多样化经验训练的鲁棒VLA模型。我们提出一种意图解耦算法，从推理上下文中分离出控制相关特征，以及一种相似样本引导的强化学习流程，以稳定策略更新并缓解分布偏移。在真实机器人任务和多模态基准上的大量实验表明，与现有方法相比，SyVLA实现了更高的任务成功率和更强的分布外泛化能力，同时有效保留了核心视觉-语言能力。代码和数据集已在项目页面发布。

## Abstract
Vision-Language-Action models face significant challenges in real-world deployment due to the entanglement of high-level reasoning with low-level control, and the instability of policy optimization. In this paper, we introduce SyVLA, a robust VLA model trained with diversified experiences. We propose an Intention Decoupling algorithm to isolate control-relevant features from reasoning contexts and a similar-sample guided RL pipeline to stabilize policy updates and mitigate distribution shift. Extensive experiments on real-world robotic tasks and multi-modal benchmarks demonstrate that SyVLA achieves superior task success rates and stronger out-of-distribution generalization compared to existing methods, while effectively preserving core vision-language capabilities. Codes and Datasets is released on \href{https://sy-vla.github.io/}{project page}.

## 1. 核心问题与整体含义（研究动机和背景）

视觉-语言-动作模型在实际部署中面临两个关键瓶颈：

- **推理与控制纠缠**：VLA 模型在模仿学习中过度强调动作学习，导致视觉-语言能力退化（灾难性遗忘），无法像人类一样通过知识迁移完成广泛任务。即使近期工作尝试混合多模态数据训练，也难以在动作能力与视觉-语言能力之间取得满意平衡。
- **策略优化不稳定**：模仿学习的目标函数（精确拟合专家动作）与真实任务成功（闭环执行数百步动作）存在显著差距。误差累积导致分布偏移，而强化学习虽能缓解，但 VLA 模型庞大的参数量和高维连续动作空间使 RL 训练极易出现策略漂移甚至能力崩溃。

因此，需要一种能兼顾动作鲁棒性与视觉-语言能力保留的 VLA 训练框架。

## 2. 方法论

### 2.1 整体架构：SyVLA

- **双系统设计**：
  - **VLM**（Qwen2.5VL-3B）：负责高层感知、语言推理和控制意图生成。
  - **Action Expert**（基于 Transformer 的 Flow Matching 模型，约 0.69B 参数）：以控制意图和多模态观测为条件，生成连续动作。
- **连接方式**：使用一组固定的可学习 **Feature Query Tokens**（20个，维度2048），追加到 VLM 自回归 CoT 输出之后。其最后一层隐藏状态（Feature Query States）经 MLP 适配器后作为 Action Expert 的条件输入。
- **三阶段训练**：
  1. **预训练**：在大规模机器人数据集（约450小时，25Hz）上混合多模态数据（比例~35%），仅约1%的机器人数据带有 CoT 标注。
  2. **任务微调**：收集数百条目标任务轨迹，进行详细 CoT 标注，混合多模态数据继续训练。
  3. **强化学习**：基于 PPO 修改，使用相似样本引导的 RL 管道。

### 2.2 意图解耦算法（Intention Decoupling）

- **动机**：推理过程的“思考”会泄漏非控制相关信息到 Feature Query States，导致动作专家决策模糊。
- **方法**：
  1. 第一次前向：用原始 Feature Query States 计算动作损失，并求其对每个 token 隐藏状态的梯度。
  2. 根据梯度 L2 范数大小，将范数低于阈值（实验中取5%分位数）的 token 隐藏状态置零（mask）。
  3. 第二次前向：使用 mask 后的隐藏状态重新计算损失并更新模型。
- **理论分析**（简化单层 Transformer 模型）：梯度范数小的 token 对应注意力分数低或值向量冗余，不贡献于控制意图，掩码后可减少非因果信息干扰。

### 2.3 相似样本引导的 RL 管道（Similar-Sample Guided RL）

- **问题**：真实世界 RL 容易因价值模型估计不准确、探索成本高而发散。
- **方法**：
  - 使用 CLIP 对观测图像编码，计算 rollout 样本与 IL 数据集样本的加权余弦相似度（三视角相机，中心权重0.83，手腕各0.083）。
  - 在每次 RL 更新时，从 IL 数据集中检索 top-K 相似样本，与当前 rollout 数据合并成 batch 训练。
  - 对 IL 样本的统一设置优势为 1.0（避免标准 GAE 导致的梯度爆炸），对 rollout 数据裁剪 GAE 到 [-0.8, 0.8]。
- **效果**：极大地稳定了策略更新，避免能力崩溃。

## 6. 主要结论与发现

- SyVLA 在真实世界任务中 **In-domain 平均成功率达 0.73**，OoD 平均 **0.64**，显著优于所有基线（包括使用十倍以上数据的 Pi0 预训练模型）。
- 在多模态基准上，SyVLA 在 AI2D、MME、HallBench 上取得最优，同时保留了可观的 VQA 能力，达到最佳平衡。
- **意图解耦算法**有效隔离推理信息与控制表示，提升动作精度；消融实验显示不使用该算法时成功率从 0.86 降至 0.43。

- 超参数 τ 在 3%~15% 范围内性能稳定，Feature Query Token 数 N≥15 即可进入稳定区域。
- 数据效率高：仅使用 Pi0 不到 5% 的预训练数据，性能接近甚至超越。
